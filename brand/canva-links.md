## Notes

- Vista Create design URLs are share links from: Vista Create → design open → Share button → "Copy link"
- Store the link to the editable design, not the download link
- Per-product thumbnail URLs all point back to copies of the Thumbnail Master — you can track just the master URL if preferred, or track each copy individually above
- Update this file as a commit each time new URLs are added: `brand: add Vista Create URLs for [asset name]`
