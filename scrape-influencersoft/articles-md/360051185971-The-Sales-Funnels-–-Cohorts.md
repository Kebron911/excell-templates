The page allows you to conduct a cohort analysis by comparing statistics data by groups that have hit [funnels](https://help.influencersoft.com/hc/en-us/articles/360051185591-Sales-Funnel) for the same period of time.

![blobid0.png](https://help.influencersoft.com/hc/article_attachments/360073105332/blobid0.png) The question mark next to the name of a step allows you to get a hint about what actions are calculated at that step.

![blobid1.png](https://help.influencersoft.com/hc/article_attachments/360073372411/blobid1.png)

## Visible fields

The **View **button allows you to add the additional data output to each step of the funnel.

![blobid2.png](https://help.influencersoft.com/hc/article_attachments/360073105352/blobid2.png)

The Amount field is displayed by default. To display other fields, tick the checkboxes next to the names of the fields.

When the **%** field is activated, the table will look like this.

![blobid3.png](https://help.influencersoft.com/hc/article_attachments/360073372431/blobid3.png)

The percentage value will be added to each step in addition to the quantitative value.

## Filter

The filter allows you to set up the output of statistics data.

![blobid4.png](https://help.influencersoft.com/hc/article_attachments/360073372451/blobid4.png)

Retrieve statistics from** ****and to** field allows you to set a period to calculate the statistics.

The most popular periods of time are contained in the quick filter at the top of the page.

The **Display**** by** button allows you to select a period for which data will be combined into one cohort.

The side filter menu contains the **Advertising** and **Tags **tabs, which allow you to retrieve statistics not from the whole funnel but just by particular parameters.

At the **Advertising **tab, you can retrieve statistics from the particular[ channel/source/campaign/advertisement/keys](https://help.influencersoft.com/hc/en-us/articles/360051550771-New-Campaign-).

![blobid5.png](https://help.influencersoft.com/hc/article_attachments/360073105452/blobid5.png)

The **Show by click** field allows you to retrieve statistics from the first and last click.

At the **Tag **tab in the field with the same name, you can specify a [tag](https://help.influencersoft.com/hc/en-us/articles/360051177191-Tags) to retrieve the statistics of the funnel.

![blobid6.png](https://help.influencersoft.com/hc/article_attachments/360073105472/blobid6.png)

The **Calculate interval analysis** checkbox opens an additional tab where you can set up the parameters of the calculation.

![blobid7.png](https://help.influencersoft.com/hc/article_attachments/360073372471/blobid7.png)

Select **the step of the funnel **you want to calculate. After that, set up the selected calculating **p****arameters **and the **interval **to sum up the statistics.

The **Cumulative data in the statistics** checkbox allows you to specify the output as a cumulative total or by individual periods.

As you select the **Interval Analysis**, the appearance of the table changes. The **Interval Analysis** block will appear with the statistics on the selected parameters with the specified interval.

![blobid8.png](https://help.influencersoft.com/hc/article_attachments/360073372511/blobid8.png)

## Data export

The **gear** button allows you to export data in the CSV format.

![blobid9.png](https://help.influencersoft.com/hc/article_attachments/360073372551/blobid9.png)

Click it and select the format in the **Export** field.
