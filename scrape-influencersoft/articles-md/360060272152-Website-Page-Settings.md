In the Website – Settings section, you can select the site and page you want to edit. To edit page settings, click on its name:

 ![mceclip0.png](https://help.influencersoft.com/hc/article_attachments/360094258152/mceclip0.png)

This page allows you to create and edit pages on the InfluencerSoft website:

![mceclip2.png](https://help.influencersoft.com/hc/article_attachments/360094258252/mceclip2.png)

[![](https://help.justclick.io/wp-content/uploads/2020/07/1595599094300.png)](https://help.justclick.io/wp-content/uploads/2020/07/1595599094300.png)

The editor page consists of 3 tabs:

- Basic parameters;

- Additional;

- Rights.

When a new page is created, it is named draft_xxxxxxxxxx. The page identifier is part of its address, so it is advisable to change it to a more meaningful version.

Note! You can use letters (A-Z, a-z), numbers (0-9), underscore (_) and hyphen (-) to specify the identifier.

## Basic parameters

### HTML editor

The main parameters tab is used to configure the page layout. To work with the page, use the [InfluencerSoft template designer](https://help.influencersoft.com/hc/en-us/articles/360050388752-Creating-and-Editing-Pages-in-the-Page-Builder) or html editor where you can add complete page code:

[![](https://help.justclick.io/wp-content/uploads/2020/07/2020-07-24_17-08-05.png)](https://help.justclick.io/wp-content/uploads/2020/07/2020-07-24_17-08-05.png)

In order for you to be able to use the HTML editor in the page settings, you need to select it at the stage of creating a page, when choosing a site template:

[![](https://help.justclick.io/wp-content/uploads/2020/07/2020-07-24_17-12-37.png)](https://help.justclick.io/wp-content/uploads/2020/07/2020-07-24_17-12-37.png)

### Timer setting

Clicking on the “Off” button opens options for setting the timer:

[![](https://help.justclick.io/wp-content/uploads/2020/07/2020-07-24_17-15-46-e1595600248402.png)](https://help.justclick.io/wp-content/uploads/2020/07/2020-07-24_17-15-46.png)

You can choose the type of timer:

-

with an absolute end time (for example, in the case of a sale that ends at 00:00 on a given day);

- with a repetition of the countdown every X minutes;

- with a relative end time, 2 options are possible here:

-

the countdown starts from the moment you enter the page;

- the countdown starts from the moment the email is sent (this option is used in an automatic email series, for example, when you offer a product with a discount to new subscribers).

Below is the page setup window “after timer expires”.

If the html editor was used, then a field for entering the html page code will be shown:

[![](https://help.justclick.io/wp-content/uploads/2022/07/2020-07-24_17-34-13.png)](https://help.justclick.io/wp-content/uploads/2022/07/2020-07-24_17-34-13.png)

If the Template Editor was used then the “Choose a template” and “+Copy” buttons will be shown:

[![](https://help.justclick.io/wp-content/uploads/2022/07/2020-07-24_17-36-39.png)](https://help.justclick.io/wp-content/uploads/2022/07/2020-07-24_17-36-39.png)

“Choose a template” allows you to create a page after the timer expires from scratch.
“+ Copy” allows you to copy an already prepared page and make changes to it.

### Split testing

Split testing (A/B testing) allows you to choose the option with the best conversion by analyzing the audience’s response to changes in any element of the page.

Note! It is not recommended to change more than 1 page element at a time, because it will not be clear which element change affected the conversion.

Clicking on the title Variant №… allows you to collapse / expand the current version of the page. This is useful for switching between page variations:

![mceclip3.png](https://help.influencersoft.com/hc/article_attachments/360094182511/mceclip3.png)

To conduct A/B testing, click on the “Add variant” button. A similar editor window will be added under the first variant, allowing you to enter the second variant of the page. Similarly, you can add 3rd, 4th variants etc.

It is recommended to copy the page from one window to the second and make changes to one element.

Note. The total value in the window should be 100% likely to be shown. Thus if we test 2 variants, we set 50%, if 3 – 33%, 33% and 34%, and so on:

![mceclip4.png](https://help.influencersoft.com/hc/article_attachments/360094182551/mceclip4.png)

## Additional

The additional tab allows you to set the code in the <head> section of the page. For example, this is where the retargeting code is placed.

Also on this tab you can enable “Protecting videos from YouTube“. If the checkbox is active, the video will not have the YouTube logo and the player’s appearance will be modified.

And if you check “Hide title of the video”, the player will also hide the title of your video.

[![](https://help.justclick.io/wp-content/uploads/2022/07/2020-07-24_17-54-37.png)](https://help.justclick.io/wp-content/uploads/2022/07/2020-07-24_17-54-37.png)

## Rights

This tab allows you to configure access to the additional administrators page:

[![](https://help.justclick.io/wp-content/uploads/2022/07/2020-07-24_17-59-24.png)](https://help.justclick.io/wp-content/uploads/2022/07/2020-07-24_17-59-24.png)

## FAQ (answers to frequently asked questions)

**How to insert the code for verifying the page into the head or the code of the metric, pixel?**
Go to the “Website” – “Pages” section. Click on the “Set up” button next to the desired site. In the site settings, select the “More” tab – and add the required code to the “Add HEAD code” field.

**How do I add a favicon (unique icon in the address field) to pages?**
Go to the “Website” – “Pages” section. Click on the “Set up” button next to the desired site. In the site settings, select the “Main Settings” tab – and select the desired image using the “Icon (favicon.ico)” button.

**Why is the page not showing?**
After creating the page, if you click on the link to this page and it does not appear (a blank white page opens), most likely, this indicates that a timer is enabled in the settings of this page, which has expired.
To solve the problem:

- Go to the settings of the page that is not displayed. To do this, go to the Websites – Pages section – click on the name of the desired site – in the list of pages that opens, click on the name of the required page.

- In the page settings, disable the timer by clicking in the page settings, in the drop-down list, on “Disabled” (the timer is located under the editor window). Then save the page settings.

**What is the timer function for in the main page settings?**
The timer function is needed so that you can set the timer for the selected page in the settings. After the timer expires, the page will be hidden and either a blank page or a page you selected will be displayed after the timer expires.
