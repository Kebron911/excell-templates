On this page, you can see the statistics for the particular period and for the selected products.

The page is available from the menu **Analytics **⇒ **Sales ****R****eport**. Also, it opens every time you log into the account.

![blobid0.png](https://help.influencersoft.com/hc/article_attachments/360073106172/blobid0.png)

As you click on the blue numbers in the Created, Confirmed, and Paid columns, it opens the [Bills](https://help.influencersoft.com/hc/en-us/articles/360050851031-Orders-) page (with the filter by date and the state of a payment).

## Chart

The** Show the chart** button expands a graphic interpretation of the main parameters at the top of the page:

-
green bar – income

-
red bar – expenses

-
green line – profit

![blobid1.png](https://help.influencersoft.com/hc/article_attachments/360073106192/blobid1.png)

## Filter

The filter allows you to set up the way the table looks.

![blobid3.png](https://help.influencersoft.com/hc/article_attachments/360073373071/blobid3.png)

Select the period and the products to display the statistics.

Data on all the products for the last months is displayed on default.

Also, you can select the type of grouping – by days, weeks, or months.

You can specify the necessary filter settings and click the **Search **button. To return to the default settings, use the **Cancel **button.
