This page allows you to create order buttons for paid products.

When you click this button, you will be taken to the Contact Information form and then to the payment page for the selected product.

When you open the page, it looks like this:

 ![blobid0.png](https://help.influencersoft.com/hc/article_attachments/360072493751/blobid0.png)

You must select the product to which the order button is created.

Click on the blue folder to select a category. Select a product by clicking on the checkbox (empty square) – it should have a blue tick.

After that, the page will take the following form:

 ![blobid1.png](https://help.influencersoft.com/hc/article_attachments/360072493771/blobid1.png)

There are four other tabs:

-
Button type

-
Form fields

-
Additionally

-
Advertising label

A preview is available at the bottom of each tab.

## Button type

Here you set the size, color, and text of the button.
 ![blobid2.png](https://help.influencersoft.com/hc/article_attachments/360072493791/blobid2.png)

![blobid3.png](https://help.influencersoft.com/hc/article_attachments/360072269792/blobid3.png)

![blobid4.png](https://help.influencersoft.com/hc/article_attachments/360072493831/blobid4.png)

You can also add a custom button by selecting “Upload your own button”.
 ![blobid5.png](https://help.influencersoft.com/hc/article_attachments/360072493851/blobid5.png)

You must click on the **Upload Image** button. In the opened window of the file manager select **Download**.
 ![blobid6.png](https://help.influencersoft.com/hc/article_attachments/360072269812/blobid6.png)

And  select which picture to use as a button.

After loading this into the file manager, the button can be re-used for other forms.

## Form fields

On this tab, you must select the fields that appear in the order form after clicking on the button.
 ![blobid7.png](https://help.influencersoft.com/hc/article_attachments/360072269832/blobid7.png)

![blobid8.png](https://help.influencersoft.com/hc/article_attachments/360072269872/blobid8.png)

Click on the corresponding checkboxes for the fields you would like included in the form.

## Additional Details

On this tab you can set the **discount coupon code by default**. It will automatically be added to the **Discount** field when the form is displayed.

If identical order forms are installed on the site on different pages then, using tags, you can track where the placement is more efficient.

In the **Tag **field, enter the label and track the statistics on the Tags page.
 ![blobid9.png](https://help.influencersoft.com/hc/article_attachments/360072493891/blobid9.png)

![blobid10.png](https://help.influencersoft.com/hc/article_attachments/360072269912/blobid10.png)

Another use of this tab is the creation of an individual button placed on a third-party partner resource.

If you specify the** partner’s login** in the field with the same name, all the customers who made the order through this button will be automatically assigned to the specified partner.

## Ad tag

This tab allows you to set parameters for tracking the effectiveness of advertising (this can be done in the Analytics section).
 ![blobid11.png](https://help.influencersoft.com/hc/article_attachments/360072493911/blobid11.png)

![blobid12.png](https://help.influencersoft.com/hc/article_attachments/360072493931/blobid12.png)

You can specify:

-
The channel

-
A source

-
The campaign

-
The Ad

-
Keys

## Checking for result

This field allows you to evaluate the result and get the code to insert.
 ![blobid13.png](https://help.influencersoft.com/hc/article_attachments/360072493951/blobid13.png)

The button is clickable. When you click on it, the product order window opens.

If everything is OK, there is the HTML code for the insertion in the **Copy this code and paste into the desired place on the page** window.
