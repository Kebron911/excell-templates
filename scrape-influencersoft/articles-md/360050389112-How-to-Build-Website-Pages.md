This page is available from the main menu of the **Websites** >> **Pages **sub-section.

It is designed to work with pages of your site in the InfluencerSoft system.

 ![blobid0.png](https://help.influencersoft.com/hc/article_attachments/360072496471)

The **Pages **section allows you to:

-
Choose a site to work with

-
View statistics for each page for the selected period

-
Add and edit pages

-
Activate, deactivate, or completely delete pages

-
Work with the [file manager ](https://help.influencersoft.com/hc/en-us/articles/360050388792)

## Selecting and configuring a site

 ![blobid1.png](https://help.influencersoft.com/hc/article_attachments/360072496491)

In order to get to work with the pages of a particular site, in the Site – Pages section, click on the name of the site that is currently needed.

Here you can also go to the settings of the site itself – assign a domain to it, add code to the header or copyright text.

You can read more about additional site settings [in ](https://help.influencersoft.com/hc/en-us/articles/360050389252-Site-Settings-)[this article](https://help.influencersoft.com/hc/en-us/articles/360050389252-Site-Settings-).

## Filter

The filter allows you to select the data to display according to the specified criteria.

 ![blobid2.png](https://help.influencersoft.com/hc/article_attachments/360072272392)

You can customize the output of pages with a specific identifier. If the entire page identifier was entered, one page will be displayed. If you entered a fragment of the page identifier – all pages that has the search term in the title will be displayed.

You can also display active or inactive pages (selected by switching the radio button).

The **Hide pages without views** checkbox allows you to remove from search results pages that did not have views for the selected period.

The **Period since… till** block allows you to configure the period for which statistics will be displayed in the table.

For convenience, the most popular periods are rendered in a quick filter at the top of the page.

 ![blobid3.png](https://help.influencersoft.com/hc/article_attachments/360072496511)

## Adding and editing pages

The **Add **button is used to create new pages.

To edit an existing page, click on its name in the **Pages **column.

## Deactivate or delete pages

To make the page temporarily unavailable, you must move the slider in the second to the last column to the left. When the color on the slider is dark gray, it means it is disabled.

To enable the page, move the slider to the right. Green shows that the page is active.

To completely delete a page, click on the X in the last column of the table.

## Statistics

Columns

Clicks/@/Sales/Conversion/Profit/Profit from a visitor

Show statistics for each page for the time specified in the filter.

The button** with the round arrows** in the last column resets the statistics for the selected period.

The **A/B **button expands the detailed statistics for each test case (if a split test was configured when creating the page).

 ![blobid4.png](https://help.influencersoft.com/hc/article_attachments/360072496531)

## View pages

The button **with the green eye** next to the page name allows you to go to the page for viewing. This will open in a new tab.

## Create sub-pages

InfluencerSoft allows you to make nested pages.

That is, if the pages have the sub-pages i.e. influencersoft.com/id_page, then the sub-pages will have the address of your login. influencersoft.com/id_pages1/id_page2

To create such pages, click on the folder next to the page name.

To return to the upper level, click on … (three dots) in the **Pages **column.

 ![blobid5.png](https://help.influencersoft.com/hc/article_attachments/360072272432)

## File manager

Each user of the system is provided with a certain amount of space for storing files.

The **File Manager** button allows you [to switch to working with the files](https://help.influencersoft.com/hc/en-us/articles/360050388792-File-Manager-) stored.

## HTML Templates

In InfluencerSoft, there are “default” page templates. However, if you want to create or upload your own unique templates, it can be done on the html-templates page. To go to it, use the button with the same name at the top of the page.
