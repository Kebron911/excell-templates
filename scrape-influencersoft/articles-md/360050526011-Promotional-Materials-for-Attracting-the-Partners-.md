/[Affiliates](https://help.influencersoft.com/hc/en-us/sections/360009245511-Affiliates) / Promotional Materials for Attracting the Partners

This page allows you to add promotional materials for attracting partners. They will be displayed in the [partner’s cabinet](https://help.influencersoft.com/hc/en-us/articles/360050391592-Partner-s-Cabinet-).

[![](https://help.justclick.io/wp-content/uploads/2016/11/77-3-1.png)](https://help.justclick.io/wp-content/uploads/2016/11/77-3-1.png)

You can add two kinds of products:

-
[Banner](https://help.influencersoft.com/hc/en-us/articles/360050390452-Drafts-for-Partners-Adding-and-Editing-an-Advertising-Banner-) (the **Add Banner **button);

-
[Text](https://help.influencersoft.com/hc/en-us/articles/360050390592-Drafts-for-Partners-Adding-and-Editing-an-Advertising-Text-) (the **Add Text **button).

Clicking the name or image in the **Material **column allows you to go to the page to edit the selected promo.

The arrows in the **Sorting** column allow you to change the order of the materials output.

Clicking the cross mark in the last column will delete the promotional material.

To return to the [Promotional Materials for Paid Product](https://help.influencersoft.com/hc/en-us/articles/360050525951-Promo-for-Affiliates-Paid-Product-Promotional-Materials-), click the** **button with the same name at the bottom of the page.
