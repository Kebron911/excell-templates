## Profile

You can set the basic settings for the user through the user profile.

The slider** “At the moment user account enabled” **allows you to activate or deactivate user access rights.

You can also upload a photo of a user here by clicking on the “Change” button.

The **Role **drop-down list allows you to select a [ready-made profile](https://help.influencersoft.com/hc/en-us/articles/360051177251-User-Role) for rights and accesses from existing ones.

***Note******:**** If the role is selected, the settings on the other tabs will not be available. Individual rights can be set only when ****no role**** is selected.*

The **Access rights **radio button allows you to set the user’s basic rights (if the role is not selected).

There are three types of rights:

-
**Call C****enter** – can change the status and comment on the call in the accounts. If you select this type, the Call Center tab will be available.

-
**Delivery** – product shipping. If you select this type, no other tabs will be available because no added settings are needed.

-
**Additional A****dministrator** – user with configurable rights. If you select this type, all added tabs will be available, except for Call Center.

Next, you need to specify the data of a specific user – first name, last name, phone number, username, email, and password etc.

A confirmation email will be sent to the indicated user’s email. The user will need to click on the link in the email to confirm access to the account.

[![](https://help.justclick.io/wp-content/uploads/2016/11/employee2.1.png)](https://help.justclick.io/wp-content/uploads/2016/11/employee2.1.png)

If you select **no role** settings (for Call Center and Additional Administrator types), you must go to the other tabs and tick the checkboxes to configure the necessary rights.

***Note******: ****Some settings are only opened when a checkmark is ticked in the top-level checkbox.*

This tab allows you to configure the rights of an additional administrator to work with the store. You can restrict a user's access to specific product(s) in the store.

[![](https://help.justclick.io/wp-content/uploads/2016/11/122.png)](https://help.justclick.io/wp-content/uploads/2016/11/122.png)

You can give access to work with all products, a certain category, or an individual product. Set up access to create and edit products, work with accounts, discounts, and co-authors.

## Site

This tab allows you to configure the rights of an added administrator to work within the site.

This tab allows you to configure the rights of an added administrator to work with the email campaign.

[![](https://help.justclick.io/wp-content/uploads/2016/11/employee3.1.png)](https://help.justclick.io/wp-content/uploads/2016/11/employee3.1.png)

You can enable [sending emails](https://help.influencersoft.com/hc/en-us/articles/360050848431-Broadcasts-) and/or [exporting subscribers](https://help.influencersoft.com/hc/en-us/articles/360050850591-Subscribers-#h_01ERAX0TXKPP463W9T49S0H43A). ## Contacts

This tab allows you to configure the rights of an added administrator to work with Contacts.

[![](https://help.justclick.io/wp-content/uploads/2016/11/2021-01-19_161920.jpg)](https://help.justclick.io/wp-content/uploads/2016/11/2021-01-19_161920.jpg)

You can give access to work with all contact lists, a certain category, or an individual list.

It is possible to [create and edit contact lists](https://help.influencersoft.com/hc/en-us/articles/360050400112-Add-and-Edit-Contact-List) and subscription forms, view [contacts](https://help.influencersoft.com/hc/en-us/articles/360050679832-Contacts) and manage [call orders](https://help.influencersoft.com/hc/en-us/articles/360051550811-Calling-Tasks).

Also, you can restrict user access to the fields that are in the contact card. By default, all fields are visible and editable. However, you can give rights to only view or to change and view.

In this case, if you give permission to “all fields,” then all those fields that are currently available, as well as those that you create in the future, will be available.

## FAQ (frequently asked questions)

-**Why is it when you add a new user's email address it is not displayed in the settings?**

You have added a new user, specified the necessary data in the settings, saved them, but the user’s email address is not displayed in the “Email” field?

When adding / changing an email address, the Email will remain the same (or it will be the account holder's email if the user is being added for the first time) until the user confirms their email address.

When changing a user’s email, a confirmation email will be sent. Subject: “**confirmation of user’s email address changing**.”
A user needs to open this email and follow the link in it. After that, the email will be confirmed and be replaced with a new one in the user settings.

**-Why can't a user log on to their account?**

If a user cannot log into his/her account, the reason in most cases is that:

-
 Invalid login link. The correct login link looks like this: [https://LOGIN.influencersoft.com/shops/bills/](https://login.influencersoft.com/shops/bills/), where “LOGIN” is the login of the main administrator of the account.
The correct link (with the already specified “LOGIN”) is in the registration confirmation letter, which is sent to the user’s mail.

-
The user enters their email in the login field, not the username. It is necessary to use the username to login.

-  The user did not confirm account registration. To confirm, the user needs to click on the link in the activation email.
