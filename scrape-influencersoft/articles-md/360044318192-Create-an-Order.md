# Create an Order

**Section:** Store
**URL:** https://help.influencersoft.com/hc/en-us/articles/360044318192-Create-an-Order
**Article ID:** 360044318192
**Updated:** 2021-05-24T06:53:03Z

---

If during a telephone call the customer expressed interest in buying a product, you can create an order using this form and send it to the customer's email.

[![](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-2.png)](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-2.png)

The Form **set up an order** consists of 3 tabs.

## Main settings

On the Basic Settings tab, you must specify the client's email, name, select the desired product and set the order cancellation parameters.

If the order formation is initiated from the [**Successful conversation**](http://help.influencersoft.com/archives/83) form, the E-mail and Name fields will already be filled out.

Clicking the Add product button opens a window with a list of products.

[![](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-2-1.png)](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-2-1.png)

You must select the product(s) to which the order is generated and click the **Accept **button.

## Customer Information

The Customer Information tab allows you to enter the necessary data into the database.

![](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-3.png)

In addition to the phone, you can specify the client's mail address in this tab, which is actual when delivering physical goods.

## Advertising label

Here you can specify the labels that will be assigned to the order.

[![](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-4.png)](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-4.png)

## Setting up orders for an employee order

Creating orders from employees' orders can be configured in such a way that a label with the manager's login will be automatically added to the order.

This setting is adjusted in the section [Store => Settings => General settings](http://help.influencersoft.com/archives/1957#i-2) by checkbox. **New orders created by managers are marked with a separate advertising channel and source**:

[![](https://help.justclick.io/wp-content/uploads/2016/11/2019-12-20_12-19-28.png)](https://help.justclick.io/wp-content/uploads/2016/11/2019-12-20_12-19-28.png)

If you set a check mark, then when creating a new order by the manager, it will not be possible to set the **Channel** and **Source**, the default channel will be set as **call-center**, and the **manager's login** will be pulled into the source:

[![](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-6.png)](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-6.png)

As a result, the order will be marked as follows:

 [![](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-8.png)](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-8.png)

And in the [Advertisement reports =>Channels](http://help.influencersoft.com/archives/1540) section you can see the statistics for the specified label:

[![](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-9.png)](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-9.png)

After all necessary fields are filled, you need to click the** Save** button to generate the order and send the invoice to the client.

## FAQ (answers to frequently asked questions)

**-How can I resend the letter with the product after paying the invoice?**

If your client has not received paid materials that should come automatically after payment, you can manually resend them. For this:

- Click on the required account number in the Shop – Accounts section.

- In the opened account card, click on the button “Resend the letter” Thank you for your purchase “.
