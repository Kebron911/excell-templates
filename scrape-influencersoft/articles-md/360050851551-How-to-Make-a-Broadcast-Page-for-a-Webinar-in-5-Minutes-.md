The functionality of InfluencerSoft has a new opportunity to set the broadcast code from YouTube directly on the page created in our service.

You can conduct free live broadcasts and manage interactive ad units during the event. Pages are also configured for mobile devices too.

Using interactive blocks, you can easily set a button to link to a proposal, and during the broadcast, turn on the button at the right time.

You can create and configure the broadcast on your page in 5 minutes.

## Create a page

First, create and prepare a page for broadcast. For this:

Go to the section “Websites/Pages” => select the desired site

 ![blobid0.png](https://help.influencersoft.com/hc/article_attachments/360072496171/blobid0.png)

On the page that opens, click on the “Add page” button.

 ![blobid1.png](https://help.influencersoft.com/hc/article_attachments/360072272052/blobid1.png)

Next, select the template you like from the “Webinars” tab (they are already configured and prepared for the webinar) and edit (change) the template in the editor.

![blobid2.png](https://help.influencersoft.com/hc/article_attachments/360072272072/blobid2.png)

You can learn more about how to create and edit a page in the template designer from [this article](https://help.influencersoft.com/hc/en-us/articles/360050388752-Creating-and-Editing-Pages-in-the-Template-Designer-).

## Interactive blocks

These will be on the page we have prepared. On this page, there are already interactive blocks for managing them during the webinar (this is a block with a button, a block with comments, a block with the name of the webinar in the page header). When you configure the page, you can either leave it or remove it if there is no need for any.

### Interactive Block Management

You can manage interactive blocks on a separate page in the “Interactive blocks management” section.
You can go to this section from the main page settings by clicking on the “Interactive blocks management” button.

 ![blobid3.png](https://help.influencersoft.com/hc/article_attachments/360072272092/blobid3.png)

You can find out more about the interactive block management section [here](https://help.influencersoft.com/hc/en-us/articles/360050389092-Interactive-Blocks-).

## Add a broadcast link to your page in InfluencerSoft

We only need to insert the broadcast link on the webinar page in InfluencerSoft.

**For this:**

-
In your personal account on InfluencerSoft, open the created webinar page in the editor.

-
Click on the video on the page with the left mouse button.

-
A blue popup will appear.

-
Click on the edit icon in this panel. ![blobid4.png](https://help.influencersoft.com/hc/article_attachments/360072496191/blobid4.png)

-  In the settings menu that opens, in the video field, delete the link that is there by default and paste the link to your broadcast on YouTube.

 ![blobid5.png](https://help.influencersoft.com/hc/article_attachments/360072272112/blobid5.png)

So, in just a few simple steps you can prepare a page for a webinar.
