## Additional lead fields

If necessary, you can specify additional fields that will be filled when [adding a new contact](https://help.influencersoft.com/hc/en-us/articles/360050677012-Add-Contact-).

To add, click the **Add Field** button and fill out the form in the pop-up window.

![](https://help.justclick.io/wp-content/uploads/2016/11/2019-10-26_18-27-09.png)

This information should be indicated in the form:

-
Field name

-
Type, which can take the following values:

Integer

-
Fractional number

-
Date

-
Date and time

-
String (up to 255 characters)

-
Text (of any length)

-
Logical type (yes or no)

-
Drop-down list

-
Switches

- The default value (you do not need to fill it in).

If the selected type is a* drop-down list*, the form will change to:

![](https://help.justclick.io/wp-content/uploads/2016/11/2019-10-26_18-29-56.png)

An additional field **Value 1** appears where you want to enter a possible value from the drop-down list. The **Add Value **button will add the next input field.

After filling all fields, click the **Save **button. The field will be added.

## Instructions for call assignments

This tab allows you to add instructions for [calling jobs](https://help.influencersoft.com/hc/en-us/articles/360051550811-Calling-Tasks)[ ](https://help.influencersoft.com/hc/en-us/articles/360051550811-Calling-Tasks)for [employees](https://help.influencersoft.com/hc/en-us/articles/360051351592-Users)[ ](https://help.influencersoft.com/hc/en-us/articles/360051351592-Users)of the call center.

![](https://help.justclick.io/wp-content/uploads/2016/11/2019-10-26_18-32-30.png)

To create an instruction, the default InfluencerSoft text editor is used. The **Save **button is used to save the changes.

## Messages for call orders

On this tab, you can set up default letters that are sent by call center employees to customers after a call.

![](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot_2019-10-26_11.png)

You need to use the radio button to select a letter template. The options include:

-
The conversation went through

-
Did not get through

-
Call back

-
Already paid

-
Invalid lead

There is a default letter for each option. However, with the default text editor InfluencerSoft, you can make changes and additions to the text.

The** Reset Default Mail** button returns the original version of the message.

## Instructions for call orders

This tab allows you to add instructions for calling [accounts](https://help.influencersoft.com/hc/en-us/articles/360050851031-Orders-)[ ](https://help.influencersoft.com/hc/en-us/articles/360050851031-Orders-)for employees of the call center.

![](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot_2019-10-26_22.png)

To create an instruction, the default InfluencerSoft text editor is used. The **Save **button is used to save the changes.
