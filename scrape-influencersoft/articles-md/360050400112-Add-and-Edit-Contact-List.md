This page allows you to add new or edit existing lists of contacts

*The pages for adding and editing are identical to each other except for the page title (Add or Edit, respectively).*

The page consists of the following tabs:

-
Basic data

-
API

## Basic data

Be sure to enter the list **name** and select the [sender](https://help.influencersoft.com/hc/en-us/articles/360050400112-Add-and-Edit-Contact-List).

![](https://help.justclick.io/wp-content/uploads/2016/11/05.png)

The remaining fields are optional.

You can:

-
Arrange lists of contacts by combining them into [categories](https://help.influencersoft.com/hc/en-us/articles/360050400732-Categories) (they must be [created](https://help.influencersoft.com/hc/en-us/articles/360050400832-Add-and-Edit-Categories) first). Also, the distribution by category can be adjusted at any time.

-
Change the text of the activation letter.

-
Set the redirect page after unsubscribing.

## API

On this tab, you can specify the address of the script, which will receive notifications when adding a new contact to the list.

![](https://help.justclick.io/wp-content/uploads/2016/11/2020-02-06_22-45-20.png)

After making the necessary changes, click the **Save **button. The list will be added to the list on the [Contact Lists](https://help.influencersoft.com/hc/en-us/articles/360050401352-Contact-Lists) page.
