The contact lists allow you to segment available contacts.

To switch to working with lists, you need to select **Contacts **from the menu on the left and then **Lists** from the opened submenu. By default, several lists are automatically created:

-
All contacts

-
All partners

-
My contacts

-
All customers

All the lists you have created yourself are also displayed on the page.

![](https://help.justclick.io/wp-content/uploads/2016/11/2020-06-05_11-18-32-2.png)

The form allows you to:

-
Add and edit lists ([normal](https://help.influencersoft.com/hc/en-us/articles/360050400112-Add-and-Edit-Contact-List), [automatic](https://help.influencersoft.com/hc/en-us/articles/360050400392-Add-and-Edit-Contact-Auto-Lists), and [for inactive contacts](https://help.influencersoft.com/hc/en-us/articles/360050861631-Adding-and-Editing-a-Group-of-Inactive-Contacts))

-
Add and edit [categories](https://help.influencersoft.com/hc/en-us/articles/360050400732-Categories)

-
Configure the output of lists by certain criteria using filters

-
Add lists to favorites

-
Enable and disable [auto-series of letters](https://help.influencersoft.com/hc/en-us/articles/360050848451-Email-Series-)

-
Test email series

-
View list reports

-
Delete lists

## Filter

The filter allows you to select lists by specified criteria.

![](https://help.justclick.io/wp-content/uploads/2016/11/2019-10-25_21-16-45-2.png)

You can select lists by:

-
A certain category

Name – if you enter the whole title, one list will be selected. If you enter a fragment, all lists where the fragment is part of the name will be selected

-
Type

-
Client – those who made a purchase

-
Information – those who are subscribed to the newsletter

It is not necessary to fill in all the fields of the filter. You must specify the filter values and click the **Search **button. The lists corresponding to the filtering conditions will be displayed in the table. To display a complete table, click the **Filter **button, and then click **Clear**.

## Adding and editing a list

To [add an ordinary list](https://help.influencersoft.com/hc/en-us/articles/360050400112-Add-and-Edit-Contact-List), use the **Add **button.

To [add an auto list](https://help.influencersoft.com/hc/en-us/articles/360050400392-Add-and-Edit-Contact-Auto-Lists) (the list to which contacts are added automatically when the specified conditions are met), use the **Auto**** L****ist** button.

For inactive contacts, you can use the Inactive button to configure a [specific auto list](https://help.influencersoft.com/hc/en-us/articles/360050861631-Adding-and-Editing-a-Group-of-Inactive-Contacts) to which the contacts will be added after a specified period of inactivity (do not open the emails or do not click on the links in the emails).

To edit an existing list, choose from the **Name **column.

## Adding and editing categories

[Categories](https://help.influencersoft.com/hc/en-us/articles/360050400732-Categories) allow you to combine several thematic lists into one array. This can be convenient if you have several areas of work that are radically different from each other, e.g., projects for the early development of children and weight loss. If you have categories, you can select all lists of contacts for the same subject with a single click.

To go to the form for [adding and editing categories](https://help.influencersoft.com/hc/en-us/articles/360050400832-Add-and-Edit-Categories), click the **Categories **button.

## Activation and deactivation of auto-series letters

You can, if necessary, disable sending an [automatic series](https://help.influencersoft.com/hc/en-us/articles/360050848451-Email-Series-) of messages configured for a list of contacts. To do this, drag the green button slider in the **Auto-series** column to the left. The button will change its color to black, and the distribution of the auto-series will be stopped.

![](https://help.justclick.io/wp-content/uploads/2016/11/2020-06-05_15-35-25.png)

To enable the auto-series, you must move the slider to the right. The newsletter will be resumed.

## My Favorites

Clicking on an asterisk in the first column of the table allows you to mark the list as a favorite. The asterisk will change its color from gray to yellow.

Such lists are displayed at the top of the table.

## Testing

The Test button allows you to send auto-series emails to the test address specified in the [mailing settings](https://help.influencersoft.com/hc/en-us/articles/360050848471-Mailing-Settings-). This will allow you to evaluate the appearance of emails in the mail client.

## List reports

Here you can see the report on the sources of the funnel and the report on the cohorts of the funnel of the selected list.

## Delete

To delete a list of contacts, click the X in the last column of the table.

## Import

To import contacts from a file into a list, click the corresponding import icon.

![](https://help.justclick.io/wp-content/uploads/2016/11/2020-06-09_10-13-31.png)

Select a file in the specified format from your computer and import the contacts into the list.

For more information on importing contacts from files, see this [Knowledge Base article](https://help.influencersoft.com/hc/en-us/articles/360050401032-How-to-Import-Leads-From-a-CSV-File).
