# Add and Edit a Series of Payment Reminders via Email

**Section:** Store
**URL:** https://help.influencersoft.com/hc/en-us/articles/360044767631-Add-and-Edit-a-Series-of-Payment-Reminders-via-Email
**Article ID:** 360044767631
**Updated:** 2021-05-24T06:20:58Z

---

The form allows you to add a new/edit an existing series of payment reminders via email.

The edit page is completely identical to the add page except for the header (Edit instead of Adding).

If it is necessary to enter/change the name of the conversation, then select the products for which the chain of reminders is created (tick checkboxes):

[![](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-27.png)](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-27.png)

Then click the **Save** button.

You will be taken to the [Payment reminders via email](https://help.influencersoft.com/archives/136) page where a new (empty) or modified series of letters will appear in the list of letters.
