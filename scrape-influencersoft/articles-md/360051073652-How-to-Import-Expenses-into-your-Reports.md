# How to Import Expenses into your Reports

/ [Reports](https://help.influencersoft.com/hc/en-us/sections/360009133772-Reports) / How to Import Expenses into your Reports

To calculate the efficiency of the advertisement, you need both the income and expense data. This page allows you to add the expense data to each advertising channel.

![mceclip0.png](https://help.influencersoft.com/hc/article_attachments/360079866032/mceclip0.png)

You should upload the expenses list according to the given pattern:

![blobid1.png](https://help.influencersoft.com/hc/article_attachments/360074195551/blobid1.png)

Or you could upload a CSV file with the same data structure.

Click the **Add **button. The expenses will be added to the **Advertising **page.

The **Expenses Editor** button allows you to change the earlier expense data.

Select the necessary channel in the filters and make changes.

![blobid2.png](https://help.influencersoft.com/hc/article_attachments/360074195591/blobid2.png)
