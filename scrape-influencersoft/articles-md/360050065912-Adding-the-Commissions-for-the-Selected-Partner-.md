# Adding the Commissions for the Selected Partner

/[Affiliates](https://help.influencersoft.com/hc/en-us/sections/360009245511-Affiliates) / Adding the Commissions for the Selected Partner

On the page you could set up the individual commissions rate for the selected partner.

*The Adding and Editing commissions pages are identical except for their names.*

[![](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-43-3.png)](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-43-3.png)

Select a paid **product** from the drop-down list.

After that, set up necessary values in the **Commision** field. It could be a fixed amount or a percentage of the product cost.

You can set up multi-level affiliate commissions, if necessary. By default there are 2 levels of commissions at the page.

[![](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-44-3.png)](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-44-3.png)

The **Save** button adds a custom commission to the Edit Commissions page.
