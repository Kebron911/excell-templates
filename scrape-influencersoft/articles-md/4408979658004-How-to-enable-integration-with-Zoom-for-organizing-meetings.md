## Zoom Integration

Go to the Profile (admin / employee) section >> Available meeting place >> click the “Connect Zoom” button

![](https://help.justclick.io/wp-content/uploads/2021/10/z1.png)

Log in to Zoom or go through a free registration if there is no account in the service [https://zoom.us/](https://zoom.us/)

![](https://help.justclick.io/wp-content/uploads/2021/10/z2.png)

After authorization, we accept the agreement by clicking on the button at the bottom of the page.

![](https://help.justclick.io/wp-content/uploads/2021/10/z3.png)

Now in the section Profile >> Time and place available for meetings, there is a logo of the Zoom service with the inscription “Connected”

![](https://help.justclick.io/wp-content/uploads/2021/10/z4.png)

### How to remove integration with Zoom

#### Removal in InfluencerSoft

Attention. You can set up integration with Zoom from several InfluencerSoft accounts, when you delete the integration on the InfluencerSoft side, the integration is deleted only with the profile in which the deletion took place. Integration will be active with the rest of the accounts.

To remove the integration with the Zoom service, go to Profile (admin / employee) >> Available time and place for meetings, click on the cross to the right of the Zoom logo.

![](https://help.justclick.io/wp-content/uploads/2021/10/z20.png)

#### Removal in Zoom

To completely remove the integration between Zoom and all linked accounts in InfluencerSoft, you must:

- Login to your [Zoom.us](https://zoom.us/) account

- Click “Advanced” – “App Marketplace”

- Click “Manage” – “«Installed Apps”

- Search for the “InfluencerSoft” app and click “Uninstall”
[![](https://help.justclick.io/wp-content/uploads/2021/10/02-is.png)](https://help.justclick.io/wp-content/uploads/2021/10/02-is.png)

## How to create an appointment

In order for users to make an appointment, you need to create one. Go to the Contacts section >> Meetings >> Create a meeting.

![](https://help.justclick.io/wp-content/uploads/2021/10/z5.png)

In the general settings, specify the name of the meeting and select the person in charge.

![](https://help.justclick.io/wp-content/uploads/2021/10/z6.png)

On the “Schedule and meeting place” tab, specify the time parameters and meeting place (by default, the meeting place is taken from the settings of the person in charge).

![](https://help.justclick.io/wp-content/uploads/2021/10/z7.png)

On the “Results” tab, you can add or remove a meeting result, change the color and text.

![](https://help.justclick.io/wp-content/uploads/2021/10/z8.png)

On the “Notifications” tab, select the sender’s name and, if necessary, edit the notification letters to the contact, after which we save the settings.

![](https://help.justclick.io/wp-content/uploads/2021/10/z9.png)

### How to add registration to a meeting in the funnel

To create a meeting, go to the “Funnels” section and create a funnel or edit the current one.

Adding an appointment “Add appointment entry” is available for the following blocks on the “Action” tab:

- Opt in

- Double opt in

- Single opt in form

- Double opt in form

- Payment page

- Payment form

Let’s take a subscription page as an example. In the editor of the subscription page block on the “Actions” tab, click “Add an appointment” and select the desired meeting from the list.

![](https://help.justclick.io/wp-content/uploads/2021/10/z10.png)

### An example of an appointment through a calendar in a funnel

When an appointment is added to the funnel, the user will be presented with a calendar when they sign in, where they can select a date and time and then schedule an appointment.

![](https://help.justclick.io/wp-content/uploads/2021/10/z11.png)

![](https://help.justclick.io/wp-content/uploads/2021/10/z12.png)

After sending the form to the user, a window with the data for entering the conference in the Zoom service will open.

![](https://help.justclick.io/wp-content/uploads/2021/10/z13.png)

By clicking on the link, the user can download the conference file and launch Zoom.

![](https://help.justclick.io/wp-content/uploads/2021/10/z14.png)

### How to edit a meeting request

After the user leaves a request for a meeting, a tab with meeting data will appear in the admin / employee account in the “Tasks” section.

![](https://help.justclick.io/wp-content/uploads/2021/10/z15.png)

By clicking on the name, task number or email of the user, a contact card will open, where there will be a link to enter a meeting with the user in the Zoom service.

![](https://help.justclick.io/wp-content/uploads/2021/10/z16.png)

To delete or edit an appointment, click on the appointment status on the right.

![](https://help.justclick.io/wp-content/uploads/2021/10/z17.png)

![](https://help.justclick.io/wp-content/uploads/2021/10/z18.png)

The meeting data will also be displayed in the Zoom service account in the Conferences >> Upcoming section, where you can delete or edit the meeting.

![](https://help.justclick.io/wp-content/uploads/2021/10/z19.png)
