## What is the purpose of categories?

A category groups several sales funnels that have the same underlying theme. This is useful when you have different sales areas and would like to organize and group them together (e.g. infant development courses and weight loss programs).

Go to Contacts>Lists>Categories:

![](https://help.influencersoft.com/hc/article_attachments/47196512504212)![blobid0.png](https://help.influencersoft.com/hc/article_attachments/360072492571)

## How to add, edit, and remove categories

To create a new category, click **Add** button at the top of the page.

To edit the existing category, click its **name** in the table.

To remove a category, click **X** icon in the last column of the desired category.

The pages to add and to edit the categories are identical, except for the page heading, which indicate “To add” or “To Edit,” respectively.

![blobid1.png](https://help.influencersoft.com/hc/article_attachments/360072492591)

Enter **the name of the category.**

Specify the sales funnels to be included into this category. It will show highlighted grayed out which lists are included.

Once all changes have been made, click **Save**. The category will appear on the list on the Category page.

## Filter

The Filter is designed to select the category by its name.

![blobid2.png](https://help.influencersoft.com/hc/article_attachments/360072492611)

If the name is entered in full, then one category will be displayed. If a part of the name is typed, then all categories with this part in their names will be shown.

Enter the filter parameters and click **Search**. The table will display the categories which match the filter parameters. To display the table in full, click **Filter** and then **Clea****r**.
