You can use UTM tags for paid traffic and track in Google Analytics which ad campaigns are most effective.

 ![blobid0.png](https://help.influencersoft.com/hc/article_attachments/360072272192/blobid0.png)

Thanks to this, you will be aware of which of your products sell well and, as a result, are the best-suited to your customers, and which are not advertised properly.

Information about the quantity of goods will be available for you for each transaction. You will also find out how much time customers take to make a purchasing decision and how many times they visit your site. You may use this information to estimate the expected amount of sales revenue.

## To retrieve order information and display it in your Google Analytics account:

-
**Register in Google Analytics and post the code on all pages of your site.
**The code must be for Google Universal Analytics. To check which version of your code (GA or GU) you have, you just need to look at the code itself: if there is a **ga.****js** file there, then it is the old one. If it has **analytics.****js**, then it is new and correct.

-
**Check that the code is installed on all pages of the site.**
a) Goto **Websites -> Set up** and select the desired site.
![blobid1.png](https://help.influencersoft.com/hc/article_attachments/360072272232/blobid1.png)
b) On the **More **tab, paste the code in the field “Add HEAD code” and save the page.
c) If the code is inserted correctly, it will be installed on all pages of the store.

-
**Create or update an existing resource to Universal Analytics in your GA account.**

-
**Enable eCommerce tracking for the resource****.**

If everything is done correctly, then information about the paid orders and the advertising channels that led to the sale will be sent to Google Analytics.

It should be noted that orders will not start at once, but gradually within a few hours everything will come back to normal.
