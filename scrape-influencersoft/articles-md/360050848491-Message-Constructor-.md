## Introduction

We are delighted to present a new service tool which is long-awaited by our clients – a Message Constructor. We hope it will simplify your work with the mailouts.

## How to create a message?

Section **“Campaigns”**, subsections **“Broadcasts”** and **“Email series”** handle all activities with the message constructor.

Switch to **“Message Constructor”** when you prepare a message.

 ![blobid0.png](https://help.influencersoft.com/hc/article_attachments/360072266452/blobid0.png)

To create a message in Message Constructor, click “Choose a template,” then the system will lead you to a page with the email templates:

![blobid1.png](https://help.influencersoft.com/hc/article_attachments/360072266492/blobid1.png)

Select the template you need.

![blobid2.png](https://help.influencersoft.com/hc/article_attachments/360072266512/blobid2.png)

If you are satisfied with the design and the look of the template, click “Create email” to go to message constructor:

![blobid3.png](https://help.influencersoft.com/hc/article_attachments/360072266532/blobid3.png)

## What are the components of an email template?

The Editor of a template constructor takes half of the screen. The control buttons are at the bottom:

-
To the left — page settings

-
To the right — adding new elements

 ![blobid4.png](https://help.influencersoft.com/hc/article_attachments/360072266552/blobid4.png)

Left panel buttons:

 ![blobid5.png](https://help.influencersoft.com/hc/article_attachments/360072491031/blobid5.png)

The page is divided into **three ****parts**: The header, main content, and footer.
Additional sections can be added to each part. Each section has widgets.

## Collection of ready-to-use sections

Large blocks of a message — **sections** — are divided into categories by the content: content, reviews, footer, products, and header. Ready-to-use sections can speed up the process of message creation.

To add an element into a message, click “Add element” in the right corner below:

 ![blobid6.png](https://help.influencersoft.com/hc/article_attachments/360072266572/blobid6.png)

Then just drag the required section to the place where this section should be dropped. This place will be highlighted in blue.

 ![blobid7.png](https://help.influencersoft.com/hc/article_attachments/360072266592/blobid7.png)

## How to add new widgets

To add a widget to the message, a section should be placed and selected (by right clicking the mouse).
You can add new widgets by clicking “Add element.” You should select the WIDGETS tab in the pop-up panel on the right by holding the required widget and dragging it to the required place in the section.
An element location is highlighted by a blue line in the same way as the section location.

 ![blobid8.png](https://help.influencersoft.com/hc/article_attachments/360072266612/blobid8.png)

## How to move the elements

To move a section or a widget in a message, click the block heading. Hold the element by left clicking the mouse and drag and drop it to the required place.

 ![blobid9.png](https://help.influencersoft.com/hc/article_attachments/360072266632/blobid9.png)

![blobid10.png](https://help.influencersoft.com/hc/article_attachments/360072266652/blobid10.png)

Widgets can be moved inside the section and between the sections. When moving a widget from one section to another, this widget will be the first on the list. When moving a widget to a new section, select this section (by left clicking the mouse) and drag and drop the widget to the required place inside this section.

## Changes in the elements’ design

Click the required element of a message — the right panel will display the settings for this element. Each element has ready-to-use styles which save your time.

 ![blobid11.png](https://help.influencersoft.com/hc/article_attachments/360072266672/blobid11.png)

Please note that in the widget “text” you can set different indents for the desktop and mobile versions of the email.
To do this, open the text widget settings and set the necessary indent values.

 ![blobid12.png](https://help.influencersoft.com/hc/article_attachments/360072266692/blobid12.png)

## How to add variables in a message

Variables typical for email can be added by clicking a “brackets” icon in Text widget:

 ![blobid13.png](https://help.influencersoft.com/hc/article_attachments/360072491051/blobid13.png)

Left click the required variable, and it will be added to an email template.

## How to create a customized email template

To create a customized email template, click the “star” in the settings panel.

 ![blobid14.png](https://help.influencersoft.com/hc/article_attachments/360072491071/blobid14.png)

The system will ask you to enter the name of the template, its description, and upload a template cover for the computer and smartphone, then the template can be saved.

When creating a new message, in case you want to use this template, select a “My templates” tab and the required template from the previously created ones.

 ![blobid15.png](https://help.influencersoft.com/hc/article_attachments/360072266712/blobid15.png)

## How to delete or to edit a customized email template

To delete or to edit a customized email template, go to My Templates from Campaigns — Broadcasts or Campaigns — Email series. Click View in the required template.

 ![blobid16.png](https://help.influencersoft.com/hc/article_attachments/360072491091/blobid16.png)

Select what exactly you would like to do: Edit or Delete.

 ![blobid17.png](https://help.influencersoft.com/hc/article_attachments/360072266732/blobid17.png)

Having edited the template, click “Save email” to save the changes.

## How to provide access to edit email template to an employee?

To make an editable email template accessible to an employee, go to “Employees” (in the drop-down menu in the upper right corner of your account), and check the box “Create new pages on sites” in profile editing in the “Site” tab. If you untick the remaining boxes, then the employee will be able to edit both the email templates and the website pages.

 ![blobid18.png](https://help.influencersoft.com/hc/article_attachments/360072491111/blobid18.png)

You have already understood that the Message Constructor follows the same principle as [Template Designer](https://help.influencersoft.com/hc/en-us/articles/360050388752-Creating-and-Editing-Pages-in-the-Page-Builder). So, if you have already tried the Template Designer, you can easily and quickly create a message.
