This page contains a report on traffic sources of a selected [sales funnel](https://help.influencersoft.com/hc/en-us/articles/360051185591-Sales-Funnel). Here, you can see all the channels that are driving traffic to the sales funnel – direct referrals, affiliate traffic, advertising, and so on.

![blobid0.png](https://help.influencersoft.com/hc/article_attachments/360073105852/blobid0.png)

The page allows you to track how many people have come to each step of the funnel.

Click the question mark next to the name of a step to get a hint about what actions are calculated at
that step.

![blobid1.png](https://help.influencersoft.com/hc/article_attachments/360073105872/blobid1.png)

## Filter

The filter allows you to set up the output of statistics data.

![blobid2.png](https://help.influencersoft.com/hc/article_attachments/360073372911/blobid2.png)

Retrieve statistics from** ****and to**** **field allows you to set a period to calculate the statistics.

The most popular periods of time (today, yesterday, this month, last month, 30 days) are contained in the quick filter at the top of the page.

The **Show by click** field allows you to retrieve the statistics by the first and last click.

The side filter menu contains the **Advertising** and **Tags** tabs, which allow you to retrieve statistics not from the whole funnel but just by particular parameters.

At the **Advertising **tab, you can retrieve statistics by a particular channel.

![blobid3.png](https://help.influencersoft.com/hc/article_attachments/360073105892/blobid3.png)

At the Tag tab in the field with the same name, you can specify a tag to retrieve the statistics of that funnel.

![blobid4.png](https://help.influencersoft.com/hc/article_attachments/360073105932/blobid4.png)

The** Read the interval analysis**** **check-box opens an additional tab where you can set up the parameters of the calculation.

![blobid5.png](https://help.influencersoft.com/hc/article_attachments/360073105972/blobid5.png)

Select **the step of the funnel **that you want to calculate. After that, set up the selected calculating parameters and the interval to sum up the statistics.

The **Cumulative data** check-box in the statistics allows you to specify the output as a cumulative total or by individual periods.

As you select the **Interval Analysis**, the appearance of the table changes. The **Interval Analysis **block will appear with the statistics on the selected parameters with the specified interval.

![blobid6.png](https://help.influencersoft.com/hc/article_attachments/360073372951/blobid6.png)

## Visible fields

The **View** button allows you to add the additional data output to each step of the funnel.

![blobid7.png](https://help.influencersoft.com/hc/article_attachments/360073372971/blobid7.png)

The **Amount** field is displayed on default. To display other fields, tick the checkboxes next to the names of the fields.

If the **%** field is activated, the table will look like this.

![blobid8.png](https://help.influencersoft.com/hc/article_attachments/360073105992/blobid8.png)

The percentage value will be added to each step in addition to the quantitative value.

## Data export

The **gear** button allows you to export data in the CSV format.

![blobid9.png](https://help.influencersoft.com/hc/article_attachments/360073106012/blobid9.png)

Click it and select the format in the **Export** field.
