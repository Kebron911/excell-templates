/[Advertise](https://help.influencersoft.com/hc/en-us/sections/360009245531-Advertise) / How to Add Instructions in the Partner's Cabinet

The **Instructions **menu item is available only if the author of the affiliate program has added it.

Otherwise, there is no entry or content to this menu item. You can add instructions under the affiliate program settings in the affiliate cabinet.

 ![blobid0.png](https://help.influencersoft.com/hc/article_attachments/360072499091/blobid0.png)

 ![blobid1.png](https://help.influencersoft.com/hc/article_attachments/360072275232/blobid1.png)

The Instructions tab would appear if the author has added it.

 ![blobid2.png](https://help.influencersoft.com/hc/article_attachments/360072275252/blobid2.png)

The page contains the text of the instruction and the link to the** promotional ****page**.
