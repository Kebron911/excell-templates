This page allows you to send an instant message to the selected subscriber groups specified in the CRM.

*Pages for adding and editing emails are identical to each other except for the page title (Add or Edit respectively).*

The page has three tabs:

-
Main settings

-
Restrictions

-
Additional settings

## Main settings

On the **Main Settings** tab, you must select the date and time when the message was sent.

***Note****! **Even though** the tool is called “instant messaging**,**”** there can be a certain time lag (**depending** on the number of recipients). In addition, the actual time of receiving a message depends on the recipient’s mailbox.*

You can prepare an instant message in advance (for example, scheduling send out to the following day). Then by clicking the **Send** button the message will be put on hold and the email will be sent at the specified time.

Further, select the desired group or group of contacts. Moving by category is done by clicking on the blue folder. Select a group by clicking on the checkbox (empty box). It should show a blue tick. You can select all the groups in the category by clicking on the checkbox of the category.

In the **Send on behalf** field, select one of the pre-added, and confirmed [senders](https://help.influencersoft.com/hc/en-us/articles/360050848471-Mailing-Settings-). Depending on the selected sender, the email will change.

The **Subject of the ****email**** **field is intended for entering the header of the message, which will be displayed in the subscriber’s mailbox.

The **Email**** Format** radio button allows you to choose in what form the email will be generated – HTML or plain text. If you select plain text, no methods of formatting the email are available, but the message is guaranteed to be displayed in the same way in all mailers. When selecting HTML, advanced editing capabilities for messages (editor and HTML-markup mode) are available.

After that, you need to insert the message text or change the existing one in the editor window. Editing text is done using the default text editor InfluencerSoft.

Data substitution variables allow you to insert individual values into the message text for each subscriber. For example, when clicking on the **Subscriber name** button, the variable **{$ name}** will be inserted in the message text, which will be replaced with the name of a specific subscriber when the message is sent, taken from the database.

Variable** Link ****to subscribe****/****unsubscribe** allows you to insert a link into the message, by clicking on which the subscriber will automatically be added from one group of subscribers to another and, if necessary, then removed from the group. This is a valuable tool that allows you to segment the subscription base.

Click on the title** ****Option number** … allows you to minimize or expand the current version of the message. This is convenient if you are using split testing (A/B testing) to switch between message variants.

 ![blobid1.png](https://help.influencersoft.com/hc/article_attachments/360072266892/blobid1.png)

### A/B Testing

Split testing allows you to choose an option with better opens, clicks, and conversion rates by analyzing the response of the audience to changing any element of the message.

To conduct A/B testing, click the **Add option** button. Under the first option, a similar editor window will be added, allowing you to enter the second variant of the message. Similarly, you can add the 3rd, 4th, and so on options.

 ![blobid2.png](https://help.influencersoft.com/hc/article_attachments/360072266932/blobid2.png)

***Note****. The total value in the window is **one hundred percent**. For example, if** we test **two** variants, we expose **fifty perc**ent**, if **three variants,** **then thirty-three percent**, **thirty-three percent,** and **thirty-four percent**, and so on.*

The **Preview **button shows a view of how the message will be displayed to the subscriber. Click the **Test **button to send a test email to the email recipient you specified.

## Restrictions

On the **Restrictions **tab, you can specify which subscriber will not receive this email.

 ![blobid3.png](https://help.influencersoft.com/hc/article_attachments/360072266952/blobid3.png)

For example, you can specify a date in the **Only signed with …** field to avoid resending the message to subscribers who have already received the information contained in the email earlier.

The** Exclude by Groups** window allows you to configure which other groups will not receive this email. For example, the message suggests buying a product at a discount. It can be received by customers who bought the product at full cost. To avoid this situation, just tick the checkbox in front of the *Customers group of the Product* … This group of subscribers will not receive the current email.

## Additional settings

On the Additional settings tab, the **count followed links from the message**** **checkbox, lets you enable or disable click-through-link counting in the message. This will allow you to consider not only the open rate, but also the conversion of the email. In addition, then in the instant mailing, you can [send emails ](https://help.influencersoft.com/hc/en-us/articles/360050385632-Sending-and-Editing-Email-by-Activity-)for the ones that passed or did not click on the link.

 ![blobid4.png](https://help.influencersoft.com/hc/article_attachments/360072491271/blobid4.png)

Here you can specify, if necessary, a personal redirect page after unsubscribing.

When you click the **Save **button, the email will be paused. To start sending it, you need to go to the** **[**Instant Mailing**](https://help.influencersoft.com/hc/en-us/articles/360050848431-Broadcasts-) page and click the submit button.
