On this page you can see the number of referrals and the page followers using your affiliate link.

![](https://help.justclick.io/wp-content/uploads/2016/11/2018-07-19_12-04-03.png)

At the end of the page, the graph of the number of referrals by day (without calculating the pages) is displayed.

By default, the data for the last calendar month is displayed.

Filters allow you to change the output settings.

![](https://help.justclick.io/wp-content/uploads/2016/11/2018-07-19_12-04-56.png)

You can set up the output for a particular page and for a particular period.

You do not have to fill out all the filter fields. Click on the **Search **button to apply the filter. To see the whole table, click on the **Clear **button.
