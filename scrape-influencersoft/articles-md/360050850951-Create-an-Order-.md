If during a telephone call the customer expressed interest in buying a product, you can create an order using this form and send it to the customer’s email.

 ![blobid0.png](https://help.influencersoft.com/hc/article_attachments/360072493511/blobid0.png)

The Form **set up an order** consists of 3 tabs.

## Main settings

On the Basic Settings tab, you must specify the client’s name, email, select the desired product, and set the order cancellation parameters.

If the order formation is started from the [**Successful conversation**](https://help.influencersoft.com/hc/en-us/articles/360050851011-Order-Management-Order-Card-) form, the E-mail and Name fields will already be filled out.

Clicking the Add product button opens a window with a list of products.

 ![blobid1.png](https://help.influencersoft.com/hc/article_attachments/360072493531/blobid1.png)

You must select the product(s) to which the order is generated and click the **Accept **button.

## Customer Information

The Customer Information tab allows you to enter the necessary data into the database.

 ![blobid2.png](https://help.influencersoft.com/hc/article_attachments/360072493571/blobid2.png)

In addition to the phone number, you can specify the client’s delivery address in this tab.

## Advertising label

Here you can specify the labels that will be assigned to the order.

 ![blobid3.png](https://help.influencersoft.com/hc/article_attachments/360072493591/blobid3.png)

## Setting up orders for an employee order

Creating orders from employees’ orders can be configured in such a way that a label with the manager’s login will be automatically added to the order.

This setting is adjusted in the section [Store => Settings => General settings](https://help.influencersoft.com/hc/en-us/articles/360050388432-Store-Settings-)[ ](https://help.influencersoft.com/hc/en-us/articles/360050388432-Store-Settings-)by checkbox. **New orders created by managers are marked with a separate advertising channel and source**.

 ![blobid4.png](https://help.influencersoft.com/hc/article_attachments/360072493611/blobid4.png)

If this box is ticked, it will not be possible to set the Channel and Source when a new order is created by the manager. The default channel will be set as **call**** ****center**, and the **manager’s login** will be pulled into the source.

 ![blobid5.png](https://help.influencersoft.com/hc/article_attachments/360072269592/blobid5.png)

As a result, the order will be marked as follows:

 ![blobid6.png](https://help.influencersoft.com/hc/article_attachments/360072493631/blobid6.png)

And in the [Advertisement reports =>Channels](https://help.influencersoft.com/hc/en-us/articles/360051073692-Advertising-The-Efficiency-of-the-Advertising-Campaign-)[ ](https://help.influencersoft.com/hc/en-us/articles/360051073692-Advertising-The-Efficiency-of-the-Advertising-Campaign-)section you can see the statistics for the specified label:

 ![blobid7.png](https://help.influencersoft.com/hc/article_attachments/360072493651/blobid7.png)

After all necessary fields are filled, you need to click the** Save** button to generate the order and send the invoice to the client.
