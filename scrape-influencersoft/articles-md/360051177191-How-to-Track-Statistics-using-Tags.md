/ [Contacts](https://help.influencersoft.com/hc/en-us/sections/360009245391-Contacts) / How to Track Statistics using Tags

This section is available from the **Contacts **menu and **Tags **submenu. It allows you to track statistics by tags (special tags).

![](https://help.justclick.io/wp-content/uploads/2016/11/1575987617186.png)

For example, the same subscription form can be installed on the main page of the site and in the sidebar. To track where the subscribers come from, you can tag each form with a unique tag. A similar label can be set when the order button is generated.

On this page, you can track how many [contacts](https://help.influencersoft.com/hc/en-us/articles/360050679832-Contacts), [subscribers](https://help.influencersoft.com/hc/en-us/articles/360050850591-Subscribers-), and [accounts](https://help.influencersoft.com/hc/en-us/articles/360050391512-Orders) were received for each tag.

Depending on the selected column, clicking on the number in the row of the table allows you to go to the Contacts page, Subscribers, or Accounts respectively, where the data will be filtered by the selected tag.

The Title tag filter allows you to display the data of those tags with the same text as in their title.

![](https://help.justclick.io/wp-content/uploads/2016/11/2019-10-26_12-28-17.png)

If the entire title is entered, one line will be displayed; if a title fragment is entered, then all tags that contain this fragment will be displayed.

You must enter the data in the filter field and click the **Search **button. To remove the filter, click the **Clear**** **button.
