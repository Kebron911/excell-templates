This page is intended for forming and editing a call-by-contact task for call center employees.

***Note!**** You can add a call request only if you have *[added](https://help.influencersoft.com/hc/en-us/articles/360050690672-Adding-and-Editing-a-User)* at least one employee with **a** Call Center profile.*

*![](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-54-1.png)*

*Pages for adding and editing jobs are absolutely identical. They differ only in the heading (Add **or** Edit).*

The page consists of four tabs:

-
Main settings

-
Statuses

-
Tips for the employee

-
Client letter draft

## Main settings

You need to fill in the following data for the call assignment:

-
**Name**

-
**Category**

-
**Responsible employee**** **– select from the drop-down list of employees with the profile type Call Center. It is a required field.

-
**Automatically assign contacts to**** the ****personal**** ****manager **– takes on the task manager to ring up.

-
**Take contacts from these ****lists**** to **allow you to select a [list](https://help.influencersoft.com/hc/en-us/articles/360050401352-Contact-Lists) for calling. If necessary, use the **Exclude contacts list** to remove contacts that are in two lists at the same time.

-
Moving categories in blocks are done by clicking on the blue folders. Select a list by clicking the checkbox; it should have a blue tick.

-
**A****dd contacts only with phone numbers **allows you to download only those contacts from the list that have phone numbers.

## Statuses

By default, there are four statuses:

-
New

-
In work

-
Successfully completed

-
Unsuccessfully completed

This tab allows you to change the possible statuses of the call assignment.

![](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-56-1.png)

Click on the status bar to change its name.

Click **Create intermediate** link to add an additional status.

![](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-57-2.png)

It can also be renamed by clicking on the name. To delete an intermediate status, click the cross at the end of the line.

## Tooltip for an employee

This tab allows you to add a description of the task and a phone conversation script for the employee.

![](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-58-2.png)

## Preparation of an email to the client

This tab allows you to edit the letter templates that are sent to contacts after the call.

![](https://help.justclick.io/wp-content/uploads/2016/11/Screenshot-59-2.png)

You can create a letter for each of the following cases:

-
The conversation went through

-
Did not answer the phone

-
Call back

-
Wrong number

-
Renouncement

To do this, switch the radio button to the appropriate template and edit the default version of the letter, if necessary. Editing is carried out using the default text editor, InfluencerSoft.

After making changes on all tabs, click the **Save **button. The task will be added to the [call history list](https://help.influencersoft.com/hc/en-us/articles/360051550811-Calling-Tasks).
