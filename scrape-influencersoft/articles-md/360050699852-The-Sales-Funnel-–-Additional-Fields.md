The current page contains a report on additional fields of the selected [sales funnel](https://help.influencersoft.com/hc/en-us/articles/360051185591-Sales-Funnel). If your contacts have [additional fields](https://help.influencersoft.com/hc/en-us/articles/360051177871-CRM-Settings), you can see a comparison of the statistics data over a selected field in this report.

![blobid0.png](https://help.influencersoft.com/hc/article_attachments/360073374271/blobid0.png)

## Filter

The filter allows you to set up the output of statistics data.

![blobid1.png](https://help.influencersoft.com/hc/article_attachments/360073107872/blobid1.png)

The **Retrieve statistics from…****to** field allows you to set a period to calculate the statistics.

The most popular periods of time (7 days, 5 weeks, 4 months, 1 year) are contained in the quick filter at the top of the page.

The **Display by the ****additional**** field** allows you to select an additional field from the list. The report will be made in this field.

The side filter menu contains the **Advertising** and **Tags** tabs, which allow you to retrieve statistics, not from the whole funnel, but just by the particular parameters.

In the **Advertising** tab, you can retrieve statistics by a particular channel.

![blobid2.png](https://help.influencersoft.com/hc/article_attachments/360073374311/blobid2.png)

In the Tags tab, in the field of the same name, you can specify a tag to retrieve the statistics of its funnel.

![blobid3.png](https://help.influencersoft.com/hc/article_attachments/360073374331/blobid3.png)

The **Read the interval analysis** checkbox opens an additional tab where you can set up the parameters of the calculation.

Select **the step of the funnel **you want to calculate. After that, set up the selected calculating **parameter **and the **interval **to sum up the statistics.

The **Cumulative data in the statistics** checkbox allows you to specify the output in a cumulative total or by individual periods.

As you select the **Interval Analysis**, the appearance of the table changes. The **Interval Analysis** block will appear with the statistics on the selected parameter with the specified interval.

## Visible fields

The **View **button allows you to add the additional data output to each step of the funnel.

[![](https://help.justclick.io/wp-content/uploads/2017/04/2019-12-16_14-38-04.png)](https://help.justclick.io/wp-content/uploads/2017/04/2019-12-16_14-38-04.png)

The **Amount** field is displayed by default. To display other fields, tick the checkboxes next to the names of the fields.

In the example above, the % field is activated, so the table will look like that.

The percentage value will be added to each step in addition to the quantitative value.

## Data export

The gear button allows you to export data in the CSV format.

![blobid4.png](https://help.influencersoft.com/hc/article_attachments/360073374351/blobid4.png)

Click it and select the format in the **Export **field.
