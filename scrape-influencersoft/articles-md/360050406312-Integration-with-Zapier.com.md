## Zapier Service Options

Zapier is a platform for transferring data between web applications using software that supports more than 1,000 applications. Zapier can be used to combine different cloud services and automation of various processes.

Zapier saves time and money allowing you to concentrate more on your business. Here are two typical problems that you will come across in an online business:

-
Searching for a programmer, setting a task, monitoring the implementation (and if a freelancer...);

-
Researching for the programmer for support and improvement.

You can set up a free profile in Zapier (will be available within 14 days after registration) and easily deal with setting up connections between applications.

This service helps to automate daily tasks and combine two or more applications.

The main Zapier features the following:

-
Synchronization with more than 1,000 services (Google, Slack, Evernote, etc.);

-
Setting triggers and actions;

-
Automatic operation;

-
Maintenance of all API;

-
Synchronization between applications;

-
Actions management;

-
Action templates;

-
Automation of repetitive tasks;

-
No need for programming knowledge.

Currently, the basic integration with Zapier is implemented in InfluencerSoft.

### How Does It Work?

During integration setup, InfluencerSoft will notify Zapier when there is a subscription to the list or when there is a sale of the product. And then you will have a huge space for choice and action, and where (in what service) to transfer this data.

In fact, it turns out that Zapier acts as a link for the integration of InfluencerSoft with other services. You do not need knowledge of the API and there is no cost for programmers.

## How to Set Up Integration

- On the opened page of the personal account ([link](https://zapier.com/app/zaps)), click the button **Make a New Zap**.![](https://help.justclick.io/wp-content/uploads/2019/07/2019-07-31_00-45-03.png)

- Next, select the application **InfluencerSoft**. Start typing “InfluencerSoft” in the search bar and you will see it appear in the search results.
![](https://help.justclick.io/wp-content/uploads/2019/07/1577558752202.png)

- After adding the application, you will see a page with the ability to select triggers. Select one of the triggers:
— **Added to List** (subscribe to list, adding the contact)
— **New Purchase** (new purchase, paid product)
— **New Lead** (triggers when a new lead is created)
— **New Order** (new order, created, but not paid)
After selecting the trigger, click  **Save + Continue**.
![](https://help.justclick.io/wp-content/uploads/2019/07/1576228367769.png)

- Specify the following information on the opened page:
1. **Login**
In this field, enter your InfluencerSoft username.
2. **API Key**
Enter the API key in this field. You can copy the key itself on the page:
https://YourLogin.influencersoft.com/shops/setts/apisettings/
You can also go to this page from any section of your personal account by clicking the link “Integration and API” in the footer of any page:

![](https://help.justclick.io/wp-content/uploads/2016/11/1576223758657.png)
3. **Domain**
Select domain in this field: **influencersoft.com**
After filling in all the fields, click “Continue”.

5. Click “Continue” again on the opened page. After that, our trigger is set up!![](https://help.justclick.io/wp-content/uploads/2019/07/1576228503406.png)

6. Next step is to add an action for the configured trigger.
To add an action, click on the link “Your Zap currently lacks Action step. Add one now!”.![](https://help.justclick.io/wp-content/uploads/2019/07/1576228574520.png)

7. A page will open where you can select the application for the action. That is, the application to which the data will be transmitted from InfluencerSoft. You will only have to select the required application and configure it to complete the integration.![](https://help.justclick.io/wp-content/uploads/2019/07/1576228662886.png)

8. After configuring, the action throws on the page where you need to enter the name of the created Zap, and be sure to include it.

![](https://help.justclick.io/wp-content/uploads/2019/07/1576228780267.png)

## What Parameters can be Transferred from InfluencerSoft to Zapier

With the integration configured, InfluencerSoft notifies Zapier, and transfers the following data:

**If the <<Added to List>> trigger is selected, this data will be sent: **

- first name (if not specified, nothing is transmitted)

- middle name (if not specified, then nothing is transmitted)

- last name (if not specified, then nothing is transmitted)

- email address

- phone

- city ​​(if not specified, nothing is transmitted)

- group name

- group id (data from group settings, “API” tab, “Group ID” field)

- group category (if not specified, nothing is transmitted)

- tags (one or more)

- customer information (data from the field of the same name in the contact card)

- contact ip

- contact time zone (UTC+03)

- contact id

- contact creation date

- global status of the contact

- lead personal manager (last name, first name, login)

**If the <<New Purchase>>or <<New Order>>trigger is selected, then when there is a sale of the product or placing an order ,the following are transmitted:**

- Order id

- Order amount

- Tax sum

- Order currency

- Order created at

- Order paid at

- Order status

- Payment method

- Order IP

- Product id

- Product type

- Product order page

- Product Name

- Product price

- Product thank you page for your purchase url

- Billing address

- Billing country

- Billing city

- Billing state

- Billing zip

- Shipping address

- Shipping country

- Shipping city

- Shipping state

- Shipping zip

- Lead id

- Lead email

- Lead first name

- Lead last name

- Lead middle name

- Lead phone

- Lead city

- Lead tags

- Lead description

- Lead ip

- Lead created at

- Order sales manager (only if the “New Purchase Trigger” trigger has been selected) – the following are transferred: last name, first name, login

- Lead personal manager (last name, first name, login)

- **Shipping/Name** – the name of the consignee in the delivery address

- **Shipping/Phone** – phone number of the consignee in the shipping address

**If the<<New Lead>> trigger is selected, then when a new contact is created, the following are transmitted:**

- Lead id

- Lead email

- Lead first name

- Lead last name

- Lead middle name

- Lead phone

- Lead city

- Lead description

- Lead ip

- Lead created at

Therefore, if there is a subscription or sale of the product, InfluencerSoft will transmit the data to Zapier. Moreover, from Zapier you can already set up data transfer anywhere.

## What Parameters can be Transferred from Zapier to InfluencerSoft

With the integration configured, Zapier transfers the following data to your InfluencerSoft account:

If the action **«Add/Update Lead» **is selected:

-
**Add to List –** Select the list to add a lead.

-
**Remove from list –** Select the list to remove a lead.

-
**Add tag –** The tags will be added to the lead. Ex. “tag1, tag2, tag3”.

-
**Remove tag –**The tags will be removed from the lead. Ex: tag1, tag2, tag3.

-
**Lead email –** Email address of the lead you want to add or update.

-
**Lead first** **name –** Name

-
**Lead last name –** Surname

-
**Lead middle name –** Middle name

-
**Lead phone –** Phone number

-
**Lead billing address 1 –** Payer Address 1

-
**Lead billing address 2 –** Payer address 2

-
**Lead billing city –** Contact city (payer, card holder)

-
**Lead billing state** – Region (region) of the contact (payer, card holder)

-
**Lead billing zip –** Contact index (payer, card holder)

-
**Lead billing country code –** ISO country code. [More details](https://en.wikipedia.org/wiki/List_of_ISO_3166_country_codes)

-
**Lead shipping address 1 –** Shipping Address 1

-
**Lead shipping address 2 –** Shipping Address 2

-
**Lead shipping city –** Delivery City

-
**Lead shipping state –** Delivery State (Region)

-
**Lead shipping zip –** Shipping Address Index

-
**Lead shipping country code** – ISO country code. [More details](https://en.wikipedia.org/wiki/List_of_ISO_3166_country_codes)

-
**Lead utc** – Timezone

-
**Lead description –** Customer information

-
**Lead personal manager –** Personal manager. Surname First name (login). All employees with the rights of a call center and additional admin are displayed. You can choose one.

-
**Lead Facebook** – Facebook profile link

-
**Lead Instagram –** Instagram profile link. Example https://www.instagram.com/jkennedy. If you transfer in @jkennedy format, then conversion to a link will occur

-
**Lead WhatsApp –** WhatsApp Profile (Number)

-
**Lead Telegram –** Link to Telegram account. Example https://t.me/jkennedy. If you transfer data in @jkennedy format, it will be converted to a link

-
**Lead Viber –** Profile (number) Viber

-
**Lead Vkontakte –** Vkontakte profile

-
**Lead points** – Points of lead

-
**Additional contact fields **([more details](https://help.influencersoft.com/hc/en-us/articles/360051177871-CRM-Settings))

The changes that Zapier makes through the “Add/Update Lead” action will be recorded in the [contact history](https://help.influencersoft.com/hc/en-us/articles/360050397772-Contact-History). A contact history note (next to the date and time) will indicate “zapier – Add / Update Lead”.

[![](https://help.justclick.io/wp-content/uploads/2019/07/2020-02-24_19-59-32.png)](https://help.justclick.io/wp-content/uploads/2019/07/2020-02-24_19-59-32.png)

**If the action <<Add Tag to Lead>> is selected:**

-
**Lead tag** – Add a tag for the contact (you can specify multiple tags separated by commas)

-
**Lead email** – Email address for which you want to add the tag (s)

**If the action «Remove Tag from Lead» is selected:**

-
**Lead tag** – The tag which will be removed from the lead. Example: tag1, tag2, tag3.

-
**Lead emai**l – Email address of the lead you want to remove a tag.

**If the action «Add Lead to List» is selected: **

-
**List** – Select a group to add a contact (sorting in alphabetical order)

-
**Lead email** – Email address to add to the group

**If the action «Remove Lead from List» is selected: **

-
**List** – Select the list to remove a lead.

-
**Lead email **– Email address of the lead you want to remove from this list.

**If the action «Unsubscribe Lead» is selected: **

-
**Lead email **– Email address to unsubscribe from all groups and set the global status of “Unsubscribed”.
