This page allows you to send an instant message to subscribers depending on the behavior (according to the activity criteria from earlier emails).

*Pages for adding and editing emails are identical to each other except for the page title (Add or Edit respectively).*

The page has four tabs:

-
Main settings

-
Restrictions

-
Additional settings

-
Testing the distribution of spam

## Main settings

On the** Main Settings** tab, you must select the date and time when the message was sent.

***Note****! **Even though** the tool is called “instant messaging” there can be a certain time lag** **depending** on the number of recipients**. In addition, the actual time of receiving a message depends on the recipient’s mailbox.*

You can prepare an instant message in advance (for example, schedule the email send out to the following day). Then by clicking the **Send **button the email will be put on hold and sent at the specified time.

Next, select the desired  activity parameter.

List of parameters:

-
The email was opened: All selected emails or any of the selected emails

-
The email was not opened: All selected emails or any of the selected emails

-
Subscriber clicked on the link in the email: In all selected emails or in any of the selected emails

-
Subscriber did not click on the link in the email: In all selected emails or in any of the selected emails

Choose the email(s) from the list the recipients of which, based on the selected activity, will form a mailing list for this email: Blue background – the email is selected, white – not selected. By default, the last five emails are displayed. **The Show 5** button opens the next five emails in reverse chronological order.

In the **Send on behalf** field, select one of the pre-added, and confirmed [senders](https://help.influencersoft.com/hc/en-us/articles/360050848471-Mailing-Settings-). Depending on the selected sender, the message will change.

The **Subject of the ****email** field is intended for entering the header of the message, which will be displayed in the subscriber’s mailbox.

The** ****Email**** Format **radio button allows you to choose in what form the message will be generated – HTML or plain text. If you select the plain text, no methods of formatting the message are available, but the message is guaranteed to be displayed in the same way in all mailers. When selecting HTML, advanced editing capabilities for messages (editor and HTML-markup mode) are available.

After that, in the editor window, you need to insert the message text or change the existing one. Editing text is done using the default text editor on InfluencerSoft.

Data substitution variables allow you to insert individual values into the message text for each subscriber. For example, when clicking on the **Subscriber name **button, the variable **{$ name}** will be inserted in the message text, which will be replaced with the name of a specific subscriber when the message is sent, taken from the database.

Variable **Link for ****subscribing**** or ****unsubscribing** allows you to insert a link into the message, by clicking on which the subscriber will automatically be moved from one group of subscribers to another without entering any data into the subscription form. This is a valuable tool that allows you to segment the subscription base.

Click on the title **Option ****Number** … allows you to minimize or expand the current version of the email. This is convenient if you are using split testing (A/B testing) to switch between message variants.

 ![blobid1.png](https://help.influencersoft.com/hc/article_attachments/360072266832/blobid1.png)

### A/B Testing

Split testing allows you to choose an option with better open, clicks, and conversion rates by analyzing the response of the audience to changing any element of the message.

To conduct A/B testing, click the **Add option** button. Under the first option, a similar editor window will be added, allowing you to enter the second variant of the email. Similarly, you can add the 3rd, 4th, and so on options.

***Note****. The total value in the window is 100%. **If **we test **two** variants, we **split recipients 50/50 between the two**, if **three **variants, then** **33%, 33%, and 34%, and so on.*

The **Preview **button shows a view of how the message will be displayed to the subscriber. Click the **Test** button on the mail to send a test email to the specified recipient.

## Restrictions

On the **Restrictions** tab, you can specify which subscriber will not receive this email.

 ![blobid2.png](https://help.influencersoft.com/hc/article_attachments/360072491231/blobid2.png)

For example, you can specify a date in the **Only signers with** … field to avoid resending the message to subscribers who have already received the information contained in the email earlier.

The **Exclude by Groups** window allows you to configure which other groups will not receive this email. For example, the message suggests buying a product at a discount. It can be received by customers who bought the product at full cost. To avoid this situation, just tick the checkbox in front of the *Customers group of the Product* … This group of subscribers will not receive the current email.

## Additional settings

On the tab **Additional settings**, the **count followed links from the message**** **checkbox, lets you enable or disable click-through-link counting in the message. This will allow you to track the number of emails opened, but also the conversion of the email. In addition, you can send emails to those that passed or did not click on the link.

 ![blobid3.png](https://help.influencersoft.com/hc/article_attachments/360072491251/blobid3.png)

Here you can specify, if necessary, a personal redirect page after unsubscribing.

After making changes on all tabs, you can click the **Send **button. Sending will begin at once, or the email send out will be scheduled and will begin at the time specified in the settings.

When you click the **Save **button, the email will be paused. To start sending it, you need to go to the [**Instant Mailing**](https://help.influencersoft.com/hc/en-us/articles/360050848431-Broadcasts-) page and click the submit button.
