/[Store ](https://help.influencersoft.com/hc/en-us/sections/360009133412-Store)/ Adding and editing product categories

This page allows you to add new or edit existing [product](https://help.influencersoft.com/hc/en-us/articles/360050388332-Products-)[ ](https://help.influencersoft.com/hc/en-us/articles/360050388332-Products-)categories.

 ![blobid0.png](https://help.influencersoft.com/hc/article_attachments/360072493391/blobid0.png)

*Pages for adding and editing categories are identical to each other except for the page title (Add or Edit respectively).*

Enter a category name.

If necessary, you can create a subcategory by selecting the top-level category in the **Parent category** field.

Specify which products are included in this category (if you need to select several products, press Ctrl + the product). If you do not select any products, an empty category will be created. Products can be added to it at the time of [creation](https://help.influencersoft.com/hc/en-us/articles/360050850851-Adding-and-Editing-a-Product-).

After making the necessary changes, click the **Save **button. The category will be added to the list on the [Categories](https://help.influencersoft.com/hc/en-us/articles/360050388292-Product-Categories-)[ ](https://help.influencersoft.com/hc/en-us/articles/360050388292-Product-Categories-)page.
