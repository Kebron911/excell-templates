/ [Contacts](https://help.influencersoft.com/hc/en-us/sections/360009245391-Contacts) / Add and Edit Categories

This page allows you to add new or edit existing categories.

*Pages for adding and editing categories are identical to each other except for the page title (Add or Edit respectively).*

## Main settings

[![](https://help.justclick.io/wp-content/uploads/2016/11/846-01.png)](https://help.justclick.io/wp-content/uploads/2016/11/846-01.png)

Enter the **name of the category** (since it will be seen by subscribers, it is advisable to make it meaningful).

Specify which [lists ](https://help.influencersoft.com/hc/en-us/articles/360050401352-Contact-Lists)are included in this category (if you need to select several lists, do this with the **Ctrl** key pressed on the keyboard). To create a new category, you need add to it at least one contact list. After that, you will be able to add new lists to the category at the time of [creation](https://help.influencersoft.com/hc/en-us/articles/360050400112-Add-and-Edit-Contact-List).

Select the [sender](https://help.influencersoft.com/hc/en-us/articles/360050848471-Mailing-Settings-) from which letters will be sent in this category.

Depending on the selected sender, the **footer of the letter** will change.

After making the necessary changes, click the **Save **button. The category will be added to the list on the [Categories](https://help.influencersoft.com/hc/en-us/articles/360050400732-Categories) page.
