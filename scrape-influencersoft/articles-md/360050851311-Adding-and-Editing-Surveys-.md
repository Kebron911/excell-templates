Pages for adding and editing surveys are identical, they differ only by name (Create or Edit Survey).

The page consists of four tabs:

-
Main settings

-
Pages

-
Actions

-
Additional settings

## Main settings

On this tab, you must specify the name of the survey and its description. In the **Description **field, for example, you can add a short explanatory instruction for the interviewers.

If you want to reach out to as many subscribers as possible with the survey, you may send an email to the database several times with a request to fill out the survey. The **Add to list** field will dedupe those who have already answered questions. You may create a separate group named “Polled,” for example, and specify it in this field. When sending a repeat reminder email for the survey, specify in the settings to exclude those who are in this group.

 On this page also, using the radio button, you can specify whether the survey will be unlimited, or if it is valid until a certain date.

![mceclip3.png](https://help.influencersoft.com/hc/article_attachments/360097209291/mceclip3.png)

## Pages

On this tab, the questions themselves are configured directly. By default, your store's primary domain is automatically associated with your survey page URL.

![mceclip16.png](https://help.influencersoft.com/hc/article_attachments/360097278772/mceclip16.png)

You can add all the questions to one page or distribute them on several pages for better readability.

### Adding a page

To add new pages to the survey, use the **Add page** button. It will appear below the first page as a similar-looking block, Page 2. If necessary, you can add more pages.

![mceclip4.png](https://help.influencersoft.com/hc/article_attachments/360097277852/mceclip4.png)

On each page, you need to specify a description. Then you can start adding questions.

### Adding questions

Clicking the **Add Question** button opens a pop-up window.

![mceclip5.png](https://help.influencersoft.com/hc/article_attachments/360097277872/mceclip5.png)

You must select the type of question. Depending on the option selected, additional fields will appear that need to be filled.

For any type of question selected, you need to provide the question itself in the Question Text field, and if necessary, specify a description. Use the checkbox to show whether the answer to this question is mandatory.

**One answer**. The answer is selected using the radio button. You must specify the answer options.

![mceclip6.png](https://help.influencersoft.com/hc/article_attachments/360097277912/mceclip6.png)

By default, 2 options are added, if you need more, use the button to** add the ****response ****option****.**

The arrows to the right of the answer options allow you to change their positions.

**Multiple answers**. The answers are selected using the checkboxes. You must specify the answer options.

![mceclip7.png](https://help.influencersoft.com/hc/article_attachments/360097277972/mceclip7.png)

By default, two options are added, if you need more, use the button **to add the ****response option**.

The arrows to the right of the answer options allow you to change their positions.

The Add option “Other” button allows you to add a text field to the answer options, where you can enter any text if none of the proposed options are suitable.

![mceclip8.png](https://help.influencersoft.com/hc/article_attachments/360097278012/mceclip8.png)

This field will always be the lowest, the X to the right of it allows you to remove this option.

**String field**. Reply with custom text with up to 255 characters. If you select the **Allow to enter only digits** check box, you cannot enter anything except digits in the field.

![mceclip9.png](https://help.influencersoft.com/hc/article_attachments/360097278032/mceclip9.png)

**The text block** allows you to enter custom-length text. It is necessary to set the block size in lines (if the size of the response in the visible part of the block exceeds the scroll bar).

![mceclip10.png](https://help.influencersoft.com/hc/article_attachments/360097278052/mceclip10.png)

## Actions

For questions with fixed answers (one or multiple answers), you can specify adding and deleting to and from groups when selecting answers.

Clicking the **Add action** button opens a pop-up window.

![mceclip11.png](https://help.influencersoft.com/hc/article_attachments/360097278072/mceclip11.png)

It is necessary to choose a question where the answer to which entails an action.

![mceclip12.png](https://help.influencersoft.com/hc/article_attachments/360097209711/mceclip12.png)

After selecting a question, additional form fields appear.

![mceclip13.png](https://help.influencersoft.com/hc/article_attachments/360097278132/mceclip13.png)

If we want to add or delete only those participants who answered the question in a certain way, then in the **Contact **field we must choose* answered*. If we want to add or delete all the participants, except those that the question in a certain way, then in the **Contact **field we choose *did not** respond*.

Next, use the radio button to select the answer.

After that, specify the group(s) to which the respondents and groups will be added, or from which they will be deleted. You can move by category by clicking on the blue folder. Select a group by clicking on the checkbox (empty square). Choosing all groups in the category can be done by clicking on the checkbox category.

Then click the **Save **button.

![mceclip14.png](https://help.influencersoft.com/hc/article_attachments/360097278372/mceclip14.png)

The added action can be deleted by clicking on the X.

## Language settings

On this tab, you can specify the default text that would be on your survey fields.

![mceclip2.png](https://help.influencersoft.com/hc/article_attachments/360097277632/mceclip2.png)

The default page looks like that:

![mceclip15.png](https://help.influencersoft.com/hc/article_attachments/360097209951/mceclip15.png)

However, you can, for example, offer a gift or bonus to everyone who takes part in the survey. Then you can make the page on which to place the reward. You can set the address of this page here.

The **Save **button saves the survey and adds it to the [Surveys](https://help.influencersoft.com/hc/en-us/articles/360050851751-Surveys-)[ ](https://help.influencersoft.com/hc/en-us/articles/360050851751-Surveys-)page.
