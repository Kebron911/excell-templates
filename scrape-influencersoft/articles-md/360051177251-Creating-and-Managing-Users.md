This page allows you to work with user role – typical rights and access settings.

[![](https://help.justclick.io/wp-content/uploads/2016/11/2020-06-15_15-38-18.png)](https://help.justclick.io/wp-content/uploads/2016/11/2020-06-15_15-38-18.png)

You can:

-
Create, edit, and delete profiles

-
Select profiles by specified parameters

-
View user activity

-
Go to work with the rights settings for an individual user

## Filter

The filter allows you to customize the output to the table of profiles that meet the specified criteria. You can search for profiles either by username, status or access rights.

![mceclip0.png](https://help.influencersoft.com/hc/article_attachments/360079864752/mceclip0.png)

If the username is entered in its entirety, one line will be displayed; if a title fragment is entered, all the profiles that contain this fragment will be displayed.

It is not necessary to fill in all the fields of the filter. You must enter the data in the filter and click the **Search **button. To remove the filter, click the **Clear**** **button.

## Users

The **Users **button will take the user to the same page where you can see the data on the rights of each individual user. Here you can add a new user, view, modify or delete and existing user.

## Adding and editing a user

To add a new user, click the **Add user** button.

To edit an existing user profile, click on its **name** in the table.

## Deleting user profile

To delete a user profile, click on the cross in the last column of the selected profile row.

## View user activity

**The Click History** button allows you to go to the page where the activity of the selected users is displayed (at what time and which pages were opened). You can filter this date intervals or the username of the specific user.
