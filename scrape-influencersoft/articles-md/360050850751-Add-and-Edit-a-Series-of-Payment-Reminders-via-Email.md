/[Store](https://help.influencersoft.com/hc/en-us/sections/360009133412-Store) / Add and edit a series of payment reminders via email

The form allows you to add a new or edit an existing series of payment reminders via email.

*The edit page is completely identical to the add page except for the header (Edit instead of Add**).*

If it is necessary to enter or change the name of the conversation, then select the products for which the chain of reminders is created (tick checkboxes):
 ![blobid0.png](https://help.influencersoft.com/hc/article_attachments/360072493151/blobid0.png)

Then click the **Save** button.

You will be taken to the [Payment reminders via email](https://help.influencersoft.com/hc/en-us/articles/360044319232-Payment-reminders-via-email-)[ ](https://help.influencersoft.com/hc/en-us/articles/360044319232-Payment-reminders-via-email-)page where a new (empty) or modified series of emails will appear in the list of emails.
