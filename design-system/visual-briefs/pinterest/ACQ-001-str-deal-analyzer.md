## Cross-pin consistency

- **Logo / wordmark position and size:** identical across all 4 — Wordmark Cormorant 28pt italic + 32pt medium, Harbor Navy on Parchment footer, x:60 y:1420 (or y:1390 where the footer starts higher). Monogram 96×96px x:876 y:1390. URL right-aligned x:940 y:1466.
- **URL bar:** `thestrledger.com/products/str-deal-analyzer`, JetBrains Mono 12pt Harbor Navy tracking 0.20em — identical.
- **Variant-A vs Variant-B distinguishability:** A pins (1, 3) use Italic Cormorant + Clay Rose accents + warning framing; B pins (2, 4) use Roman Cormorant + Muted Gold accents + outcome framing. Test thumbnail diff before sign-off.
- **60pt minimum primary headline:** met on all 4 (60–76pt range).
- **One specific claim per pin:** Pin 1 = "AirDNA-median trap, 25th-percentile fix," Pin 2 = "verdict pills + 10-minute promise," Pin 3 = "Y1 vs stabilized side-by-side numbers," Pin 4 = "chaos → DEAL pill."
- **Verdict pill family:** appears on Pins 2 and 4 (per task SKU note that the verdict pill is iconic). Same dimensions (280×80–120px, 16px corner radius), same fill % (12% Clay Rose / Parchment Alt / 20% Muted Gold-soft), same JetBrains Mono Bold mono-spaced label.
- **Brand signatures coverage:** italic "The" (all 4 wordmarks); gold terminal period (all 4 headlines); 48px gold rule (all 4); tracked uppercase mono eyebrow (all 4).
- **Color ratio per `brand-decisions.md` §4.2:** within tolerance on all 4. Pin 3 is the only Navy-dominant pin (band + table borders); justified by Infographic archetype.
