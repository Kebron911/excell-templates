## Cross-pin consistency

- **Logo / wordmark position and size:** identical across all 4. Wordmark Cormorant 32pt, x:60 y:1430 on Parchment footer (Harbor Navy on Parchment); on Pin 2's Parchment-only footer this is also Harbor Navy. Monogram 96×96px, x:876 y:1410 across all 4.
- **URL bar:** `thestrledger.com/products/damage-claim-aircover-log`, JetBrains Mono 12pt, tracking 0.20em, right-aligned x:940 y:1438. Identical.
- **Variant-A vs Variant-B distinguishability:** A pins use Italic Cormorant + Clay Rose data callouts; B pins use Roman Cormorant + Muted Gold callouts. Test thumbnail diff (240×360px) before sign-off — the warm/calm contrast must read.
- **60pt minimum primary headline:** met on all 4 (60–96pt range).
- **One specific claim per pin:** Pin 1 = "31% scrambled," Pin 2 = "78% recovery," Pin 3 = "12 items / 14 days / 5 scripts," Pin 4 = "31% → 78% gap = $1,400 per $3,000 claim."
- **Brand signatures coverage:** italic "The" (all 4 wordmarks); gold terminal period (all 4 headlines); 48px gold rule (Pins 1, 2, 3, 4); tracked uppercase mono eyebrow (Pins 1, 2, 3 eyebrows + Pin 4 BEFORE/AFTER labels).
- **Color ratio:** Parchment-dominant pins (1, 3, 4) ~55% Parchment / 30% Navy / 8% Graphite / 4% Gold / 3% Clay Rose. Pin 2 (challenger) inverts to ~45% Navy / 50% Parchment / 4% Gold / 1% Graphite — within `brand-decisions.md` §4.2 tolerance.
