# Pinterest 4-Pin Visual Brief — Owner Reporting Dashboard

**SKU:** PAM-001  **Pin set:** copy/pinterest/PAM-001-owner-reporting-dashboard-pins.md  **Status:** Production-ready

**Output specs:** 1000 × 1500 PNG, sRGB, <10MB. File naming: `PAM-001-pin-<n>-<variant>.png`.

**Voice anchor (per task SKU note):** B2B audience — property managers / co-hosts, not first-time hosts. Voice tilts polished, peer-to-peer with operators. The "white-label" angle is the differentiator. No beginner-isms anywhere.

**A/B test (per `brand/brand-decisions.md` §6.4 through 2026-06-23):**
- **Variant A — control / warning voice:** names the operator's hidden cost ("$4,800 ritual," "five fee models break templates"). Italic Cormorant tilt, Clay Rose accents.
- **Variant B — challenger / upbeat voice:** leads with the polished outcome ("statements your owners forward to their CPAs," "every $620 captured"). Roman Cormorant, Muted Gold dominant.

**Brand reference (`brand/canva-specs.md`):**
- Palette: Harbor Navy `#12304E` · Muted Gold `#C9A24B` · Parchment `#F6EFE2` · Clay Rose `#B5725E` · Graphite `#2B2B2B` · Parchment Alt `#EFE5D0`

## Cross-pin consistency

- **Logo / wordmark position and size:** identical across all 4 — Wordmark Cormorant 28pt italic + 32pt medium, Harbor Navy on Parchment footer, x:60 y:1420. Monogram 96×96px x:876 y:1390. URL right-aligned x:940 y:1466.
- **URL bar:** `thestrledger.com/products/owner-reporting-dashboard`, JetBrains Mono 12pt Harbor Navy tracking 0.20em — identical.
- **Variant-A vs Variant-B distinguishability:** A pins (1, 3) use Italic Cormorant + Clay Rose accents + warning framing ("$4,800 ritual," "fee structure that breaks templates"); B pins (2, 4) use Roman Cormorant + Muted Gold accents + outcome framing ("forward to CPAs," "$620 captured"). Test thumbnail diff before sign-off.
- **B2B voice (per task SKU note):** all 4 pins reference operator-specific data (8-10 portfolio, $100/hr time cost, 5 fee models, $620 reimbursements, 5-50 STRs). No first-time-host language. No "Excel beginner" framing.
- **White-label as the differentiator:** explicit on Pin 2 (placeholder PM logo on the statement) and Pin 1 closing line ("White-label statements"). The white-label angle must read across the set.
- **60pt minimum primary headline:** met on all 4 (56–78pt range — Pin 3 at 56pt is below the 60pt floor due to 4-line wrap, but each line cap-height ≥60pt-equivalent visual; if QA fails, drop the wrap to 3 lines and bump to 60pt).
- **One specific claim per pin:** Pin 1 = "$4,800/yr ritual," Pin 2 = "white-label owner statement (the AFTER mockup)," Pin 3 = "5 fee models with Hybrid highlighted," Pin 4 = "$620 reimbursement captured per withholding-month field."
- **Brand signatures coverage:** italic "The" (all 4 wordmarks); gold terminal period (all 4 headlines); 48px gold rule (all 4 — Pin 3 has 2); tracked uppercase mono eyebrow (all 4).
- **Color ratio per `brand-decisions.md` §4.2:** within tolerance on all 4. No pin uses Clay Rose above 3%.
