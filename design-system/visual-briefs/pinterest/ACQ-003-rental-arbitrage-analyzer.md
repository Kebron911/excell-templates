## Cross-pin consistency

- **Wordmark:** Cormorant 400 italic 22pt + Cormorant 500 38pt with Muted Gold terminal period, Harbor Navy on parchment, centered x:500 y:1340. Width ≥160px.
- **Mono URL:** `thestrledger.com` JetBrains Mono 14pt Graphite tracked 0.22em centered x:500 y:1290.
- **SKU watermark:** `ACQ-003 · v1.0` JetBrains Mono 11pt Muted Gold tracked 0.20em, top-right x:940 y:60 — identical on all 4.
- **Eyebrow:** JetBrains Mono 14pt Muted Gold tracked 0.22em–0.30em, baseline y:140 (±20).
- **Gold rule:** 48px × 1px Muted Gold `#C9A24B`, ≥1 per pin.
- **Set echo:** Pin 1's Clay Rose `Y2 ↑` tick → Pin 2's Stress Test cell border → Pin 3's `−$400` cash flow → Pin 4's white-label proposal border. Clay Rose is the throughline that says "the part most arbitrage spreadsheets miss."
- **Palette discipline:** 6 hexes only. Clay Rose appears in all 4 pins but only as a single small accent each (tick, border, single number, border) — total surface across the set well under §4.2's 3% per-pin budget.
- **Type discipline:** Cormorant Garamond / Inter / JetBrains Mono only.
- **Brand-signature audit:** Italic "The" (all 4 wordmarks) ✓ · Gold terminal period (all 4 headlines) ✓ · 48px gold rule (all 4) ✓ · Tracked uppercase mono (all 4 eyebrows) ✓.
- **Thumbnail check:** At 200×300 thumb. Pin 1 = `Year 2` (italic) is the read. Pin 2 = the eight-cell grid pattern. Pin 3 = `+$1,400` / `−$400` opposing colors are the read. Pin 4 = the proposal doc shape with the gold-period title.
- **Voice consistency:** A pins (1, 3) name the failure / the gap. B pins (2, 4) lead with the framework / the artifact. Don't soften A. Don't warn in B.
