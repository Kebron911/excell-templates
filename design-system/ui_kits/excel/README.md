# Excel Workbook UI Kit

A hi-fi browser recreation of The 2025 Ledger — the flagship Schedule E workbook. This is NOT a functional spreadsheet; it's a pixel-perfect preview of the product the user downloads, so the product page and marketing can show exactly what they're getting.

## Files
- `index.html` — Workbook view with tab bar, formula bar, cover sheet, and income/deductions/depreciation tabs
- `components/Workbook.jsx` — main shell (ribbon, name box, formula bar, tab strip)
- `components/Sheet.jsx` — grid renderer

Styling is parchment-warm, not Microsoft-blue — the workbook matches the brand, not stock Excel.
