# Pinterest Pin Templates

Five brand-styled 1000×1500 pin templates for The STR Ledger. Pinterest pins live or die on scroll-stopping typography — every pin here earns its stop with the four brand signatures (italic "The", gold period, gold rule, tracked mono).

## Files
- `index.html` — gallery showing all five pin styles side-by-side in a Pinterest-feed layout
- `components/Pin.jsx` — the five pin components (`BigNumber`, `ListPromise`, `Quote`, `Comparison`, `StackedWords`)

## Reference
Pin copy concepts drawn from `Kebron911/excell-templates/copy/pinterest/pin-catalog-first-30.md`.

## Style notes
Pinterest algorithm favours tall, text-heavy, high-contrast pins. Every template here is 1000×1500 (2:3), uses the display face at 80–140px, and keeps a single clear claim above the fold.
