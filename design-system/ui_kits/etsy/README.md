# Etsy Storefront UI Kit

Hi-fi recreation of the **@thestrledger** Etsy shop — built against Etsy's current (2026) shop-page chrome with The STR Ledger brand dropped in.

## Files

- `index.html` — shop homepage: Etsy global nav + shop banner + owner block + announcement + listings grid
- `listing.html` — single listing detail page (TAX-001 Mileage Log)
- `components/Etsy.jsx` — shared chrome (Etsy top nav, shop nav, listing card)

Listing copy lifted from `Kebron911/excell-templates` repo:
