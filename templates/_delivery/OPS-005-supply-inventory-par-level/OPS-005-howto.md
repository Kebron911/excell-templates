# How to Use Your Supply Inventory + Par-Level Restock Calculator

## What this workbook does

Tracks consumables (linens, toiletries, cleaning supplies, paper goods, kitchen basics) per property with a *par level* (the minimum on-hand). Compared against your current count, it auto-flags below-par items on a single restock list. Eliminates the "ran out of TP between guests" failure mode, the "ran out of laundry detergent at 8pm Saturday" failure mode, and the "the cleaner emailed me at 10pm asking where the soap is" failure mode.

## First-time setup (15 min)

1. Open tab **Property Roster** — list every property.
2. Tab **Inventory** — for each property, every consumable with:
   - Item name, category, supplier, unit (rolls / bottles / packs), bulk vs individual
   - Par level (the minimum count you want on-hand)
   - Current count (you'll update this weekly or after each turnover)
   - Lead time from supplier (1 day for Amazon, 7 days for Costco run, etc.)
3. Pre-loaded with 80+ standard STR items: TP, paper towels, hand soap, dish soap, dishwasher pods, laundry detergent, dryer sheets, shampoo, conditioner, body wash, toothpaste, lightbulbs, batteries (AA + AAA + 9V), trash bags (kitchen + bath), Ziploc, foil, parchment paper, salt, pepper, sugar, coffee, tea, cleaning chemicals, cleaning rags, mop heads, sponges, dish soap, fabric softener.

## Weekly check (or after every turnover)

1. Open the Inventory tab for the property.
2. Update the Current Count column (cleaner can do this on a tablet during turnover).
3. Open Restock List — only items below par show up.
4. Order from each supplier in batches.

## Reading the outputs

- **Restock List:** auto-filtered to items < par. Sort by supplier for efficient ordering.
- **Bulk-vs-Individual Compare:** for each item, shows the $ delta between buying individually vs in bulk (Costco, Sam's Club, Amazon Subscribe & Save). Saves $200-800/year per property.
- **Lead Time Warning:** any item where (current count) ≤ (par × lead time / typical use rate) flags red — order TODAY or you'll run out.

## When you add a new property

Duplicate the Inventory tab template, change the property tag, set par levels (typically same as your other properties), zero out current counts, walk the property to fill in the actual inventory. ~30 minutes per new property to seed.
