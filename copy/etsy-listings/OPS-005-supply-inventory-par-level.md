## Description

```
Never run out of TP between guests. Per-property consumables tracker with par-level auto-restock for Airbnb / STR hosts. Eliminates the 10pm cleaner emergency. Excel + Google Sheets.

Your cleaner texts at 10pm Saturday: "I'm at the property, no laundry detergent."
Or: "We're out of paper towels and the next guest checks in at 3pm tomorrow."
Or: "There's no soap in either bathroom."

These calls happen because nobody's tracking the consumables. Maybe you do a
mental count when you visit, but mental counts forget batteries, dish soap, and
the third spare lightbulb.

This workbook ends those calls. 80+ pre-loaded items per property. Par level for
each (your minimum on-hand). Current count. Anything below par auto-flags on a
single restock list, sorted by supplier so you can knock out the order in one
batch.

═══════════════════════════════════════════
WHAT'S INCLUDED
═══════════════════════════════════════════

📋 Per-Property Inventory (80+ pre-loaded items)
   · Paper goods (TP, paper towels, napkins)
   · Bath consumables (hand soap, body wash, shampoo, conditioner, toothpaste)
   · Cleaning chemicals (all-purpose, glass, bath, dish soap, dishwasher pods)
   · Laundry (detergent, dryer sheets, fabric softener, stain spray)
   · Kitchen pantry (salt, pepper, sugar, coffee, tea, cooking oil, foil)
   · Lightbulbs + batteries (every common size)
   · Linens (sheets, towels, pillowcases — counts not consumables but tracked here)

📋 Par-Level Auto-Flag
   · Below par = appears on Restock List
   · Lead-time-aware: warns BEFORE running out (current count ≤ par × lead time / use rate)

📋 Restock List Output
   · Single sortable list, only items below par
   · Group by supplier (Amazon, Costco, Target, Sam's Club)
   · One-click ordering workflow

📋 Bulk-vs-Individual Compare
   · For each item: $ delta between buying individually vs. in bulk
   · Saves $200-800/year per property

📋 Cleaner Workflow Tab
   · Tablet-friendly view your cleaner can update during turnover
   · Quick-count column (no math, no formulas needed by cleaner)

═══════════════════════════════════════════
WHO THIS IS FOR
═══════════════════════════════════════════

· Airbnb / VRBO hosts with 1+ properties
· Multi-property operators standardizing supplies across the portfolio
· Hosts who've gotten the "we're out of X" emergency call
· Anyone who's ever wasted a Sunday on a Costco run that could've been one Amazon order

═══════════════════════════════════════════
HOW IT WORKS
═══════════════════════════════════════════

1. Open the workbook. Add properties to the Roster.
2. For each property, set par levels (typically uniform across portfolio).
3. Walk the property to seed Current Count (~30 min one-time per property).
4. Cleaners update Current Count weekly during turnovers.
5. Open the Restock List anytime — only below-par items. Order by supplier in one batch.
```
