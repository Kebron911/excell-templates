## Description

```
Rehab dollars to revenue lift, line by line, contractor by contractor. Detailed pre-launch rehab budget with ROI projection for Airbnb / STR investors. Catches the 30-50% over-budget trap. Excel + Google Sheets.

Most rehabs go 30-50% over budget. The buyer didn't have a line-item list — they
had a guess. The contractor said "I can do it for $40K"; six months later, after
"discovered" plumbing, "had to" upgrade the panel, and "while we're at it"
re-tiling, you wrote checks for $63K.

This workbook is the line-item list that prevents it. Estimate, actual, variance,
contractor, payment status — every renovation task tracked from demo day to
ribbon-cutting. Plus an ROI projection so you know whether the renovation actually
pays for itself in revenue lift.

═══════════════════════════════════════════
WHAT'S INCLUDED
═══════════════════════════════════════════

📋 Line Items (200-row capacity)
   · 50+ pre-loaded common rehab tasks (paint, flooring, kitchen, bath, exterior, electrical, plumbing, HVAC)
   · Estimate, actual, variance per row
   · Variance >25% over flags red

📋 Contractor Directory
   · Every bidding contractor with categories, payment terms, contact info
   · Payment status per draw (1099-NEC tracker integration via TAX-003)

📋 Contingency Burn-Down
   · Track buffer usage as the project progresses
   · Below 50% remaining = warning
   · Below 25% = stop, re-budget

📋 ROI Projection
   · Per category: estimated revenue lift (kitchen reno → +$15/night ADR)
   · Payback period in months
   · Total rehab → total revenue lift dollar-multiple

📋 Final Cost Summary
   · Print-ready basis-step-up record (you'll need this for depreciation)
   · Feeds TAX-013 Depreciation Tracker for the asset register

═══════════════════════════════════════════
WHO THIS IS FOR
═══════════════════════════════════════════

· STR investors planning a pre-launch renovation
· BRRRR-to-STR converters with a 60-90 day rehab window
· Hosts upgrading an existing property between seasons
· Anyone whose last rehab went 30%+ over budget

═══════════════════════════════════════════
HOW IT WORKS
═══════════════════════════════════════════

1. Open BEFORE demo starts. Load every planned task into Line Items.
2. Set total budget + contingency % on the Property Snapshot tab.
3. As bills come in, update Actual column. Variance auto-flags.
4. Update Contingency Burn-Down weekly.
5. At ribbon-cutting: print the Final Cost Summary. Save with deal folder.
6. 6 months later: re-run the ROI tab with actual revenue. Tune future rehabs.
```
