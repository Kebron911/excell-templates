## Description

```
Did PriceLabs / Wheelhouse / Beyond actually earn their monthly fee? A/B test framework for paid pricing tools. Run the tool 60 days, run without it 60 days, workbook computes the actual revenue lift vs. monthly fee. Excel + Google Sheets.

PriceLabs costs $20/mo per listing. Wheelhouse $15-25. Beyond similar.
On 4 listings that's $80-100/month or $1,000+/year. The tools all
claim "average customer sees 15-30% revenue lift." That's marketing
math from the people selling the tool — not your specific market.

This workbook is the A/B test that answers honestly. 60 days with
the tool active. 60 days without. Subtract tool fees. Statistical
confidence flag for sample size. Verdict: "tool earned $X above its
fee" OR "tool cost more than it earned."

═══════════════════════════════════════════
WHAT'S INCLUDED
═══════════════════════════════════════════

📋 Property Roster + tool settings (which tool, monthly fee, integration date)
📋 Test Plan Generator (60 days each side, season-aligned)
📋 Period 1 (Tool Active) — weekly metrics: ADR, occupancy, RevPAR
📋 Period 2 (Tool Inactive) — same metrics
📋 A/B Comparison Tab
   · Period 1 vs 2 ADR / Occupancy / RevPAR delta
   · Net revenue delta
   · Subtract tool fees
   · Statistical confidence flag (small sample warning)
📋 Verdict — keep / cancel / extend test

═══════════════════════════════════════════
WHEN THE TOOL WINS
═══════════════════════════════════════════

· High-volatility markets (NYC, LA, beach towns with weather demand)
· Multi-property portfolios where manual pricing isn't tractable
· Markets with frequent large events the tool catches you'd miss

═══════════════════════════════════════════
WHEN THE TOOL LOSES
═══════════════════════════════════════════

· Stable seasonal markets (mountain cabins with predictable peak/off-peak)
· ≤2 properties (manual pricing via REV-002 + REV-003 wins)
· Markets where you're already disciplined about REV-003 weekly comp tracking

═══════════════════════════════════════════
WHO THIS IS FOR
═══════════════════════════════════════════

· Hosts currently paying for a pricing tool (validating the spend)
· Hosts considering one (de-risk the trial)
· Multi-property operators evaluating per-property tool ROI separately

═══════════════════════════════════════════
SAMPLE-SIZE CAVEATS
═══════════════════════════════════════════

A 60-day test on a single property has noisy data. The workbook flags
low-confidence results. For a definitive answer: run multiple properties
or run year-on-year comparisons (same season this year vs last year).
```
