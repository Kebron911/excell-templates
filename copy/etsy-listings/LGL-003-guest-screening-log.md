## Description

```
Banned at property A. Can't book property B. Portfolio-wide. Guest screening log + ban list for Airbnb / VRBO hosts. Per-booking checklist + cross-property ban-list lookup so problem guests can't slip through. Excel + Google Sheets.

That guest who threw a party at your beach house — is now trying to book
your mountain cabin under a slightly different name. Without a portfolio-
wide ban list, you might accept the booking. Then it's a damage claim,
police call, possible AirCover dispute, and a 1-star review you'll respond
to publicly forever.

This workbook is the cross-property block. Every booking gets a screening
checklist. Every banned guest gets logged with the reason. The next
booking inquiry is automatically cross-checked against the ban list.

═══════════════════════════════════════════
WHAT'S INCLUDED
═══════════════════════════════════════════

📋 Property Roster + screening criteria per property
📋 Per-Booking Screening Checklist
   · Verified ID? Profile age? Prior reviews + rating?
   · Screening service result (Autohost / RoomR / manual)
   · Auto-flag if guest matches Ban List
📋 Portfolio-Wide Ban List
   · Guest name + profile URL
   · Property where incident happened
   · Date + reason (party / damage / harassment / fraud / other)
   · Documentation links (photos, police report, Airbnb resolution case)
📋 Cross-Property Block Detection — banned guest can't book any of your other listings

═══════════════════════════════════════════
WHO THIS IS FOR
═══════════════════════════════════════════

· Multi-property hosts (2+ listings)
· Hosts in markets with party-risk demographics
· Operators who've ever had a "déjà vu" booking that turned out to be a banned guest
· Anyone whose CPA / lender / insurance asked for guest-screening documentation

═══════════════════════════════════════════
LIMITATIONS
═══════════════════════════════════════════

Airbnb / Vrbo limit how much guest data they expose pre-booking. You can't
always see prior reviews on a new booking. Some screening services charge
per-check ($1-5) and pull deeper data. The workbook standardizes whatever
data you have access to.
```
