## SEO notes (Etsy algorithm factors)

Etsy ranks listings by:
1. **Keyword match** (title, tags, description opening) — covered above
2. **Conversion rate** — monitor first 30 days, iterate if < 2%
3. **Recency** — renew listing every 4 months to boost freshness signal
4. **Shop-level quality** — fast ship (automatic for digital), good reviews, complete shop profile
5. **Customer satisfaction** — reviews, low refund rate

Post-launch iteration:
- Weeks 1–2: check stats daily
- Weeks 3–4: if CTR < 1%, rewrite title (test 2 variants via Etsy Ads A/B)
- Month 2: if conversion < 2%, rewrite thumbnails (biggest conversion lever)
- Month 3: if sales < 10/mo, consider repricing down to $12 or up to $24 to find sweet spot
