## Description

```
Underwrite three Airbnb / STR deals at once. Pick the winner per metric. Side-by-side comparison spreadsheet for vacation rental investors choosing between properties. Excel + Google Sheets.

You're sitting on three offers. One has the best location, one has the best price, one
has the best cap rate. Which one do you actually buy? Running each through a
single-property analyzer separately works — but it's hard to see the relative
pros / cons at a glance, and you end up staring at three browser tabs.

This workbook drops three deals into the same view. NOI, cash-on-cash, cap rate,
break-even occupancy, IRR — all three properties, side-by-side, with a winner-per-metric
highlight strip and a BUY / WALK / NEGOTIATE verdict per deal.

═══════════════════════════════════════════
WHAT'S INCLUDED
═══════════════════════════════════════════

📋 Three Property Tabs (A / B / C)
   · Same input fields as our STR Deal Analyzer (ACQ-001)
   · Price, financing, projected revenue, OpEx
   · Each tab fully self-contained for offer-day editing

📋 Comparison Tab (the headline)
   · Three columns side-by-side
   · 14 metrics rolled up: NOI, CoC, cap rate, break-even, IRR, total cash invested, monthly cash flow, debt service, OpEx ratio, more
   · Winner-per-metric highlight strip — green = winner per row

📋 Verdict Tab
   · Per-property: BUY / WALK / NEGOTIATE based on your thresholds
   · Summary recommendation: which property to make the offer on first

📋 Sensitivity Column
   · Re-runs each property under a 10% revenue drop scenario
   · Surfaces which property is most resilient if STR market softens

═══════════════════════════════════════════
WHO THIS IS FOR
═══════════════════════════════════════════

· STR investors with 2-3 properties under serious consideration
· Hosts comparing in-market vs. out-of-state deals
· Buyers choosing between turnkey, rehab, and arbitrage opportunities
· Anyone whose realtor sent them three "great deals" and they need to pick

═══════════════════════════════════════════
HOW IT WORKS
═══════════════════════════════════════════

1. Open the workbook. Walk into the Property A tab.
2. Plug in price, financing, comp-driven revenue forecast, OpEx. Repeat for B + C.
3. Open the Comparison tab — all three side-by-side.
4. Read the Winner strip. Read the Verdict. Make the offer.
```
