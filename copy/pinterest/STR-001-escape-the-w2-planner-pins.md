# Pinterest Pin Set — STR-001 Escape the W2 Planner

**Design spec**
- Dimensions: 1000 × 1500 (Pinterest standard vertical)
- File format: PNG, <10MB
- Title font ≥ 60pt — readable at thumbnail size
- Brand colors only: Harbor Navy `#12304E`, Parchment `#F6EFE2`, Muted Gold `#C9A24B`, Clay Rose `#B5725E`, Graphite `#2B2B2B`
- Wordmark or monogram bottom-right (small, subtle), URL `thestrledger.com` mono-tracked above logo
- Italic "The" + gold period on every wordmark instance
- 48px gold rule under every headline
- Eyebrow: tracked uppercase mono, Muted Gold

**Voice note (STR-001 specific):** emotional stakes are higher than the rest of the catalog. Voice is slightly more reflective, less transactional — but still editorial. No "fire your boss," no "financial freedom," no hustle vocabulary. Frame around freedom and agency, not escape. The product is calmer than the genre.

**Style variants legend** — Tip-list · Quote-card · Infographic · Question · Before/after

**A/B brand split (per brand-decisions.md §6.4, active through 2026-06-23)**
- **Variant A (control / warning voice):** names the gap or the blocker first, then the resolution
- **Variant B (challenger / upbeat voice):** leads with the calm outcome, capable framing, no hype
- Two pins each, alternating. Same design, same destination URL — only headline + first description line differ.

## Hashtag strategy

Five max per pin. Always include `#airbnbhost` or `#shorttermrental` + topic-specific (`#financialindependence`, `#earlyretirement`, `#strinvesting`). Never `#fireyourboss`, `#hustle`, `#bossbabe`, `#9to5escape`, or any guru-adjacent tag — these violate §6.2 and §6.4 voice rules.

## Board targets

- **Hosting Like a Business** — primary for reflective/strategic pins
- **Short-Term Rental Business** — primary for analytical pins
- **STR Templates & Tools** — secondary, product-feature pins

## Tracking

Log pin-level performance weekly in `ops/pinterest-ab-test.md`. Pin ID, variant (A/B), campaign (strategic), impressions, outbound clicks, saves, outbound CTR. Decision date 2026-06-23 per brand-decisions.md §6.4.

For STR-001 specifically: monitor save rate as a secondary signal. The emotional stakes mean Pinterest users are more likely to save the pin for later than click immediately. A high save / low CTR pattern is still a win on this SKU.
