# Pinterest Pin Set — FIN-002 Break-even Occupancy Calculator

**Purpose:** Drive own-site traffic to `thestrledger.com/products/break-even-occupancy` (Etsy fallback URL `etsy.com/listing/<id>` once live). 4 pins seeded into the Financial campaign track. Scheduled via Creasquare, 1 pin/week for 4 weeks.

## Design spec

- Dimensions: 1000 × 1500 (Pinterest standard vertical)
- File format: PNG, < 10MB
- Title font on image: ≥ 60pt (readable at thumbnail)
- Brand colors: ~55% Parchment background, 30% Harbor Navy band, 8% Graphite type, 4% Muted Gold accent (period + rule), 3% Clay Rose (highlight only)
- Logo: monogram bottom-right corner, 120px square, subtle
- Italic "The" + Muted Gold period on every wordmark
- 48px Muted Gold rule under headline

## Style variants legend (per pin-catalog spec §C2)

- **Tip-list** — bulleted promise + headline
- **Quote-card** — bold claim + subtle supporting line
- **Infographic** — visual breakdown (chart, table preview)
- **Question** — provocative question headline
- **Before/after** — spreadsheet preview with overlay

## A/B brand split (per brand-decisions §6.4, active through 2026-06-23)

- **Variant A (control / warning voice):** leads with the cost or risk. §6.1 baseline.
- **Variant B (challenger / upbeat voice):** leads with the clean outcome. Same data, same product, calmer voice.
- 2 pins each, alternating.
